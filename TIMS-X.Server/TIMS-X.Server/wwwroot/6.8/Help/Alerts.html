    Alerts

    TIMS Alert System can be used to notify users of arrived appointments, tasks to be completed, or simply quick messages from other users in the same office.

    Appointment Alerts

    When an appointment for the logged in user has the status changed to one designated to "Send Alerts", the user will receive a popup alert notifying them that the patient as arrived.  This alert will popup no matter which link the user is working in.

    • Click OK to dismiss the message
    • Click View to see the Appointment Details
    • Click Snooze to dismiss the message and have it re-appear in the amount of time specified in user preferences (default is 30 seconds)
    • If the user wishes not to be disturbed by Appointment alerts, they can be turned off my accessing the Alert menu from any link. Right-click on the Alert icon in the upper right hand corner of the screen to display the menu, then un-check "Enable Appointment Alerts".  Re-check the same line to turn the alerts back on.

     

    Task Alerts

    When a new task is created for the logged in user, the user will receive a popup alert notifying them of the new task. This alert will popup no matter which link  the user is working in.

    • Click OK to dismiss the message
    • Click View to see the Task Details
    • Click Snooze to dismiss the message and have it re-appear in the amount of time specified in user preferences (default is 30 seconds)
    • If the user wishes not to be disturbed by Appointment alerts, they can be turned off my accessing the Alert menu from any link.  Right-click on the Alert icon to display the menu, then un-check "Enable Task Alerts".  Re-check the same line to turn the alerts back on.

    Message Alerts

    • To send a simple inter-office message to one or more logged in users, left-click the Alert icon in the upper right hand corner of the screen. 

    • A blank alert message screen will popup.

    • Select one or more Recipients (required)
    • Enter something in Title, 25 characters maximum (required)
    • Enter a longer message if desired
    • Click Send or Cancel to abandon changes

    • The message alert will popup on the recipient's computer, no matter which link they are working in.  It will indicate who sent the message.

      • Click OK to dismiss the message
      • Click Reply to send a return message to the sender.

      • Click Snooze to dismiss the message and have it re-appear in the amount of time specified in user preferences (default is 30 seconds)

    Disable Alerts

    To disable alerts - right click on the Send Alert button in the upper right hand corner.  From the menu, check or un-check the type of alert you wish to disable.
