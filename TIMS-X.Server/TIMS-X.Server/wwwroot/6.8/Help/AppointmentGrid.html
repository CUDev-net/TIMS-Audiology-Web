    Appointment Grid


    • The Appointment Grid displays appointments based on the View Tab chosen and criteria selected in the Filter.  
    • Collapsing the Month, Summary, Block Legend, and Patient panels allows wider columns on the Scheduler. The panels of the Scheduler can be docked, sliding or hidden. When the panels are hidden and collapsed, they appear as tabs along the sides of the screen. (see example)  Clicking on the tab of a hidden panel will reopen the panel and bring it into view on the screen with the pin horizontal. When the pin is horizontal, the panels become sliding and will be hidden again when the mouse is moved anywhere else on the appointment screen.

    • Appointments shown on the grid include all appointments made by other computers within your office. This view is updated every 60 seconds by default (but can be changed by Support if needed). New appointments made by other offices will only appear after each office replicates (generally once an hour).
    • Appointments are color-coded by provider (see example), which is displayed on a small box on the left side of the appointment box.  The provider color is assigned in Setup - Providers.

    • New in version 6.07 - Appointment status can be assigned an icon. In Appointment Status setup, choose the icon you wish to appear on the grid for appointments with that status.

    • The main block of the appointment is also color coded by Appointment Type, Site, or Resource depending on the setting in User Preferences. 

    • The description will include the Appointment Type, the Site and the Resource (if one has been selected).  In addition, if the mouse is hovered over the appointment, a tool tip will display additional details. 

    • You can choose which appointments of a certain status will display in the grid (i.e. you can set canceled appointments to not display on the grid) See Appointment Status setup.
    • The resolution (increments) of the calendar can be set and/or changed in User Preferences.   
      • If your resolution is set to 15  or 20 minute increments, you may not be able to see the whole day at once.  Use the scroll bar and/or up and down arrows on the right side, to see the time slot you need.
    • Daily Office Start and End times are also set in Site Setup.  Provider daily hours are set in User Preferences. This only effects the background color of the calendar.  Office hours are displayed in a light color, while non-office hours (and weekends) are darker.  You can still make an appointment outside of your office hours if you wish, although you will receive a popup warning asking if this is what you wish to do.

    • In the Provider View, Schedule blocks will be displayed in the color chosen when the block was setup. Schedule blocks are a way to designate certain times and days for particular appointment types and providers.  The Block Legend panel lets you know which colors signify which type of appointment block. See Setup Schedule Blocks for more information.

    • You can change the Start or End time of an appointment by dragging the block on the grid.  Hold your mouse over the top or bottom edge until a double-headed arrow appears.  Then hold the left mouse button down while dragging to the desired time slot. Let go of the mouse button and you'll then be asked if you wish to save the change to the start/end time.  Say Yes.
    • Appointment details can be viewed from the grid two ways:
      • Click once on the appointment - the details will be displayed in the Appointment Summary Panel to the right.  
      • Double-click on the appointment - the Appointment Details popup will be displayed.  Make any changes needed and click the OK button.



    Filter Calendar View

    • The Filter icon gives the users the ability to filter the calendar view according to various parameters including Providers, Sites, Resources and Specialties (Audiology or SLP).
    • The calendar view is specific to each user and any changes made do not affect other user views.
    • Clicking on the Filter icon opens a pop-up screen with the available parameters to use to search for an appropriate appointment time.

    • A maximum of 20 members of any group (Provider, Site, Resources) may be selected at any time.
    • The Resource options are only shown when a specific Site is checked and are not shown when All Sites are checked.
    • Click on the Apply button to change calendar view based on chosen parameters.
