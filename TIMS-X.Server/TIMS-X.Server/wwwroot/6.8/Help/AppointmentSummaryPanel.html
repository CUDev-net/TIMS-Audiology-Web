    Appointment Summary Panel

    Single click on an appointment on the Calendar to view the Summary window with the details for that appointment.  In the Summary Panel, the information can be viewed, but not edited.


    Below are descriptions of the appointment fields shown on the Summary.  To edit the information, double-click on the appointment in the grid or from the Patients List Panel. 


    Patient

    The last name, first name, and middle initial of the patient as entered in the Patient details. If the patient has any Communication Restrictions, the name will

    appear in red.

    Phone

    The primary phone number of the patient, as entered in the Patient details. 

    Reports button

    Click  this button to run reports for the selected appointment and/or patient.

    Intake Sheet

    Click here to quickly print a patient Intake Sheet for the selected appointment.  This can be the stock Intake Sheet, or a custom one made for your practice.

    History

    Click this button to go to the History record that was automatically created with this appointment.  This button will only be active starting on the day of the appointment or if it has already occurred.

    Provider

    The provider with whom the appointment has been scheduled.

    Appt Type

    The type of appointment the patient has been scheduled for.  

    Site

    The location where the patient appointment is taking place. 

    Resource

    The resource (room) where the appointment is taking place (if selected). 

    Status

    The current status of the appointment.

    Marketing

    The reason the patient made an appointment for that day.  If this is the first appointment for the patient and a marketing source was set in the Patient details, that source will default here.  It can be changed if needed.  May be required, depending on the practice setup.

    Date

    The date that the appointment is scheduled for.  This date will automatically be filled out when the patient’s name has been dropped onto the Appointment Grid

    Time

    The time the appointment is scheduled to start and end.  This time will automatically

    be filled out when the patient’s name has been dropped onto the Appointment Grid.  

    Duration

    The time length the appointment has been scheduled for.  This will automatically

    be set to the default duration setup for the Appointment Type that has been selected.

    Next Contact

    The date the patient is to be contacted next. This field can be used to generate

    recall mailings.

    Notes

    Free form field for notes about the patient or the appointment.  If the patient has any Communication Restrictions, those restrictions will automatically be pasted into the Notes when a new appointment is made.

    Patient Notification

    If the office has been setup to process Appointment Notifications - a manual notification (Verification and/or Confirmation) can be performed here.  See Appointment Notifications for details.

    Notification Statuses

    If any notifications have already been sent for this appointment, they will be noted here.

    Recurrence

    If the appointment has been designated as a recurring one, details will appear here. Otherwise it will display "Once".  See Recurring Appointments for details.

    Tracking


       Created

    The date when the Appointment information was originally created, along with the Site and User who created it. It cannot be edited.

       Updated

    This date and User name is automatically updated when the record has been changed and saved.  It cannot be edited.
