    Battery Club Activity List

    Standard Version:

    The Battery Club Activity List can be accessed several ways:

    • From the Patient Activity panel, double-click on the Battery Club line in the Activity panel. Then click the History button . 

    • From the HAHistory summary or Patient Summary panel, click the orange Battery Club button .  Then click the History button . 


      • Date - the date each action was taken
      • Action - Join, Pickup, or Close
      • Qty - number of batteries
      • User - user who performed action
      • Site - site where action was performed
      • Invoice (Standard w/ Invoice only) - if an invoice was created, button will be labeled View Invoice.  Otherwise the Create Invoice button will allow you to create one.


    Deluxe Version:

    The Battery Club Activity List can be accessed several ways:

    • From the Patient Activity panel, double-click on the Battery Club line in the Activity panel, and the activity list will popup.

    • From the HAHistory summary or Patient Summary panel, click the orange Battery Club button .  Then click the History button . 

    • Close the Activity List by clicking the "X" in the upper right had corner


      • Action Type - Join, Pickup, Mail out,  Add (to Balance), or Close
      • Action Date - the date each action was taken
      • Packages - the number of battery packages involved, whether added or received
      • POS Invoice - This will show the corresponding POS Invoice number, if the action (Join or Add to Balance) has been invoiced in the POS module.
      • QBInvoice (not used at this time)
      • Date Voided - if the action has been voided, the void date will be shown here.


    Void a Battery Club Action


    Standard Version:

    Select the line in the Activity List.  Right-click on the line and click Void.

    A confirmation message will popup.  Click Yes to Void, No to cancel.


    Deluxe Version:

    Double-click on the line in the Activity List.  The Battery Club screen will popup.  Click the Void button .  

    Note: You can not void a Join action, an action that has been invoiced, an Add to Balance action that will cause the balance to be negative, or a Close action.
