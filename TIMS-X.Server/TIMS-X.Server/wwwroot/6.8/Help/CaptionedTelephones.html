    Captioned Telephones

    Forms can be generated from within TIMS for three different telephone captioning providers.  From the Patient Summary Panel, click the  Reports/Letters/Captioned Telephone icon . Then click on the Captioned Phones button. 

    (If this button is not visible, contact TIMS Support at 800-763-0308 to adjust a system setting to turn it on.)  Choose the desired Service Provider:

    The More Information link will connect to the selected provider's website.

    Hamilton CapTel

    • Provider will default to the patient's default provider, but can be changed if needed.
    • Choose the Certifying Professional type and patient's Phone Preference.
    • The Outreach Educator field will default to whatever has been entered in Site setup, but can be changed/entered manually.
    • The form will default to printing the practice address, unless Use Site Address is checked.
    • Click the Next button to generated the completed form.
    • Click the Save icon to archive a copy of the generated form, or the Print icon to print a copy.
    • Check the consent box and click Submit. (see example)

    • You will receive a confirmation that the application has been electronically submitted.

    CaptionCall

    • Provider will default to the patient's default provider, but can be changed if needed
    • The Promo Code (Site) field will default to whatever has been entered in Site setup, but can be changed/entered manually.
    • Choose the Profession type, preferred Language, and patient's phone preference (Phone or IPad App).
    • The form will default to printing the practice address, unless Use Site Address is checked.
    • Click Preview to view a snapshot of the generated application form.  From here you can click the Save icon to archive a copy of the generated form.
    • Click Email to email the application form directly to CaptionCall.
    • Click Print to print a copy of the form. 

    ClearCaptions

    • Provider will default to the patient's default provider, but can be changed if needed.
    • Choose the Profession type.
    • The form will default to printing the practice address, unless Use Site Address is checked.
    • Click Preview to view a snapshot of the generated application form.  From here you can click the Save icon to archive a copy of the generated form.
    • Click Email to email the application form directly to ClearCaptions.
    • Click Print to print a copy of the form.
