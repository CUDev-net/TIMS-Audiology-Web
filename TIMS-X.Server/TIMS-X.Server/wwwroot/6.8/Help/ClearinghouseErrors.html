    Clearinghouse Errors

    Error messages received from the clearinghouse will appear in the Clearinghouse Notes field. In the case of data errors, the clearinghouse will reject the entire claim, as opposed to an individual transaction line. Therefore, all transactions that were included on that particular claim will be rejected and will have the same error message displayed in the Notes field. The clearinghouse will reject claims if either required data has not been entered or if the data entered is incorrect or invalid in some way.


    Clearinghouse error messages will have 5 sections: Error Message, Field Number, Sequence #, Field Name, and Data in Error.

    • Error Message - The description of the error. 
    • Fld: - The field containing the data field that is in error or is missing information.
    • Seq: The sequence number of the field containing the data in error 
    • Field Name: The name of the field containing the data that is in error 
    • Data in Error: The data causing the error in the claim. If missing information has caused the error, this will be blank.
