    Copying Reports

    A convenient way to customize your diagnostic report, is to copy an existing sample template supplied by TIMS and make your changes to that, instead of starting from scratch. You can then make basic formatting changes (logo, addressing, margins, font style, etc).  You can also edit tables, remove rows you don't need or add additional fields.  This is generally a much easier approach then starting with a blank page. Once you have one report with the basic format that you use, you can copy that one to additional reports. To use the Copy Patient Report function:

    1. Select the template that most closely matches the style you want to duplicate.
    2. Click the Copy Patient Report icon.

    1. The Report information screen will popup.

    1. Enter a new Name for the new report. Change the Description, Auto Archive, and/or Editable at launch settings if desired. See below for definitions. The yellow fields are required. 
      1. Enter a Name for the new letter.
      2. Change the description if desired, it will default to the copied report description.
      3. The Letter Category will default to the copied letter's category, usually Diagnostic Reports. If needed, change to SLP Reports.
      4. Auto Archive After Print will default to the copied letter's setting, but can be changed as needed. This defines how or if the letter will be archived in the patient activity after it is printed.  Archive choices are: Ask User, Always Archive, or Never Archive.
      5. Editable at launch - check this box to be able to edit the report when it is run for a selected patient in History. You will be able to change/add/remove text or graphics, change formatting, etc. and those changes will be printed and/or archived with the letter.  When this box is checked, edits can also be saved as a draft, which means you can return to the same report letter and finish the edits. You will also be able to track user changes to the drafts.  If unchecked, the user will not be able to make any changes at run time.
      6. Inactive - check this box to make the report Inactive. (a report must be made inactive before it can be deleted)
    1. When the required information is entered, click the green Save button to create the new template. Click Cancel to abandon all changes. The copied template will appear in the designer. Proceed to Edit the report as described in Editing Reports.
