    Copying a Claim

    New in version 6.08 - an easier way to submit corrected claims has been added.  You can now "copy" a claim directly in adjudication, make necessary changes, combine CPT codes if necessary, and add other required codes without having to switch screens.


    Copying a claim will leave the existing claim in adjudication.  A correcting (CR) adjustment will need to be made on each transaction line so that the patient's balance and the sales amounts will not be inflated.  The new (copied) claim will then be used going forward.


    To copy a claim follow these steps:

    1. Select the desired claim on the patient tree and click the Copy button in the Transfer section at the bottom of the Adjudication screen.  

     

    1. The Copy Claim screen will popup:

    1. Choose a Claim Frequency Code and enter the Payer Control number

    1. All of the claim lines will be Selected by default.  If you wish to only copy certain lines to the new claim, un-check the Select box on the lines you do not want to include.
    2. Double click on a line to open the Claim Transaction detail screen.  Here you can change any of the line details, if necessary. (CPT Code, Price, Diagnosis Code(s), etc).  You cannot change the price if the claim line is from an insurance transfer.
    3. Two insurance transfer lines can be combined into one line if necessary.  This would be necessary if the original claim had lines for 2 hearing aids using monaural codes.  The payer may need instead to see them as one line using a binaural code. 
      1. Combine will only be an option if the claim lines have been previously created as an Insurance Transfer and both lines have used the same CPT Code.
      2. Check the Combine box for both lines then click the Combine button.
      3. A message will popup reminding you to use a binaural code in the new claim. Click OK

      1. Only one line will now be displayed.  Double-click on it to change the CPT Code to a binaural one. 
    1. When all changes are made, click OK to copy the claim.
    2. The new claim will appear in Processing-Claims Mode.  The billed payer will be last payer billed from the copied claim. It can be further edited and/or transmitted from there.
