    Create a New Line Item

    1. Click the Add button in the Line Commands section
    2. Choose the Type of item from the first drop down field.

    1. The Item list will be based on the category chosen above.  Choose the desired item from the list.

    1. The default description will fill in the line once the item is chosen.  This can be edited if necessary.
    2. Quantity will default to 1, and can be changed as necessary.
    3. The Unit Price will default to what was entered in Setup.  If the user has permission, this can be changed as necessary.
    4. The Taxable check box will be checked if the item has previously been setup to be taxable in Setup. 
    5. If the item is to be billed to insurance, and a previous Appointment has been selected, the option to Expand and Enter Billing Codes will be visible.

    Click the down arrow, then the Enter Billing Codes button . Select the proper CPT code from CPT Codes, Modifier(s) if needed, and up to 4 Diagnosis Codes. Click OK. The Insurance Amount will default to the amount of the item, but can be a lesser amount if necessary (can not be more). See Insurance Transfers for more information.

    1. Click OK to save the line information.


    To add more lines, follow the steps listed above.
