    Creating and Editing Letters

    You can create your own letter templates by entering the exact text, graphics,  and merge fields that you need.  Once a custom letter template has been created and saved, you can edit as needed.

    Letter Panel Icons

    • Refresh Letter List
    • Add New Letter
    •  Edit Letter
    •  Copy Letter
    •  Reset Stored Parameters   
    •  Delete Letter (letter must be inactive first)
    • Display letter templates that have been marked inactive
    • Show all types of letter templates for both Communications (bulk) and Single Patient. Templates will be listed with a P (Single Patient), and/or C (Communications)

     

    Creating Letters

    1. Select a category from the Letters panel, then click the Add New Letter icon at the top of the panel.  The category chosen is only for organization on the tree.  You will be able to add information from various categories in one letter.

    1. The Letter information screen will popup.

    1. The yellow fields are required.  
      1. Enter a Name for the new letter.
      2. If this letter will be emailed to patients, you must enter an Email Subject.  This will be displayed in the subject of all the emails sent out.
      3. Availability indicates whether the letter will be used for a Single Patient, or for a bulk letter generated in the Communications link.  If the letter is created from the Communications page, that box will be checked by default.  You can also additionally assign it as a Single Patient letter. 
      4. The Letter Category will default to the category in the tree that was highlighted when you clicked the Add New icon. This can be changed here if needed. Category choices are Patient, Appointment, History, HA History, or Mailing Label. 
      5. Auto Archive After Print defines how or if the letter will be archived in the patient activity after it is printed.  Archive choices are: Ask User, Always Archive, or Never Archive.
    2. When the required information is entered, click the green Save button to create the new letter. Click Cancel to abandon all changes.
    3. A blank template will appear in the designer.  The designer functions much like Microsoft® Word and other word processor programs.

      1. The font and other formatting tools are available in the Home tab. (shown above)
      2. Change the margins, page size, column layout, etc in the Page Layout tab.

      1. Insert tables, images, headers and footers, and signature fields in the Insert tab. Hint: You can use the table function to create forms where the alignment of fields is especially important. 

        1. New in version 6.08: multiple signature fields can now be entered on one document for the the patient or provider to sign. For example, an agreement form can be designed that requires several spots for initials and a full signature. Signatures would only be used for Single Patient letters. To collect signatures, you must have a supported Topaz® signature pad. (contact TIMS Support for details) .  Position your cursor where you'd like insert the signature, then click the Insert tab, then Signature Field. You can resize the field to whatever size needed.  (larger for a signature, smaller for initials, for example). Other formatting (text wrapping, size, alignment) can be done by right-clicking on the merge fields.

      1. Also new in version 6.08: Check boxes and date pickers can be added to a letter/form that can be dynamically clicked while the letter is on the screen. The checked box and/or date selected will then printed and/or saved with the archived letter.

     

      1. Change how you view the letter in the View tab. For example, change the zoom, hide/view table gridlines and rulers, etc.

    1. To add the merge fields from TIMS, click on the tab for the type (source) of field you wish to add.  For example, click on Patient Fields to see all the fields associated with the patient. The fields are separated by category to make it easier to find the one you need.

    Position your cursor where you would like to insert the merge field, select a source tab, category, then a field from the drop down.  Click the plus sign to add the merge field to the letter.

    There are ten queries or "sources" available from which to choose merge fields, all based on different activities and available in the corresponding tabs.

      1. The majority of the letter merge fields will probably come from the "Patient Fields" tab. This is where you will find the basic demographics such as name, salutation, address, current hearing aids, referral source, practice information, etc.
      2. You can choose fields from multiple sources in one letter.  For example, you can pull the patient's demographics from the Patient Fields tab, appointment details from the Appointment Fields tab, and history details from the HistoryFields tab, all in one letter.
      3. Click on the following Source names to see a list of the available merge fields in each.
        1. Patient
        2. Appointment
        3. History
        4. SLP
        5. HA History
        6. Battery Club
        7. HA Loaner
        8. HA Order
        9. HA Return
        10. HA Repair
    1. As you're editing the letter, you can click on Merge from the Home tab at any time to preview it with sample data in the Letter Preview tab.  This can be helpful to check size of images, merge field placing and formatting, etc. Click the Email Preview tab to see how the letter will look when sent as an email.

     


    1. When you have finishing creating your letter, click the File tab, and then Save.


    1. The letter will now appear in the Letters panel, under the chosen category, and is ready for use.

    Copying Letters

    If you have a basic format for letters that you use, (logo, addressing, margins, font style, etc) - an easy way to create new letters based on that format is to use the Copy Letter function.  

    1. Select the letter you wish to copy in the Letters Panel.
    2. Click the Copy Letter icon .
    3. The Letter information screen will popup, showing summary information for the original letter.  

    Change the name to the new letter.  Change any other required data as needed - Email Subject, Availability, Letter Category and/or Auto Archive. 

    1. Click the Save button to create the new letter. The new letter will now appear in the Letters panel, under the chosen category, and appear in the designer.
    2. Make any changes needed, including text, graphics, formatting, or merge fields.
    3. When you have finishing editing the new letter, click the File tab, and then Save.

     

    Editing Letters

    Once it has been saved, you can edit an existing letter to make any changes necessary. You can change the letter summary information such as the Name and category. You can also change the letter itself, including formatting, text, and/or merge fields.

    1. Select the letter you wish to edit in the Letters Panel.
    2. Click the Edit Letter icon .
    3. The Letter information screen will popup, showing summary information of the letter. 

    Change any of the fields as needed - Name, Email Subject, Availability, Letter Category and/or Auto Archive. If the letter will no longer be used, you can also check the Inactive box at this time.  If only the summary information needs to be changed, click the green Save button to exit. 

    1. If you need to edit the letter itself - make any changes needed, including text, graphics, formatting, or merge fields in the designer.
    2. When you have finishing editing the letter, click the File tab, and then Save.

    Designer Tips and Tricks

    General

    • Merge fields need to be deleted one at a time. You can’t select multiple fields to delete at once.
    • When setting margins, the number input needs to have a leading zero.  For example "0.5"  not ".5". 
    • To increase the size of a check box field, highlight the box and change it to a larger font size.
    • If words seem to be splitting excessively on lines (over-hyphenating), you can change the settings (per user). Right-click on a misspelled word, click Options, then Hyphenation to increase the number of characters.  Other spell-checker options can be set in Spell Checking Options.

    Images

    • Inserted image bookmarks (signatures, logos, audiograms, etc.) should be formatted as “Embed data in document”. This is the default setting, but it can be double-checked here: Right-click – Format, then Saving Options, Saving Style.

    • Text wrapping for images: when images are used in tables, it works best to use Top and Bottom or In Line wrapping.

    • Image merge fields sometimes format better if a space is added after when inserted in a table.
    • Vertical text boxes are not supported in the designer. 

    Tables

    • If grid lines are not displaying properly, it may help to remove then re-add them
    • For multi-line merge fields such as notes, diagnosis, etc. - uncheck "Allow row to break across pages" to keep the field together. You need to click the box once to select, then again to uncheck. Right-click - Format Table - Size and Formatting - Options.

    • If a table is used in a header, and the cell is right-justified, it works best to increase the cell margins to 0.2

    Hyperlinks and QR Codes

    • To insert a hyperlink into an email communication, place your cursor where you would like it then from the Insert tab click Hyperlink - Insert Hyperlink.

    • Enter the text to display and the location for the link to go to.  It can either be a URL (website address) or an email address.

     

    • It will appear as a clickable link on the document. 
    • A scannable QR code can be added in a similar way.  From the Insert tab, click Barcode-QRCode.

    • Once the code appears on the document, right-click and select Format.  From the Type and Color tab, enter the URL of the website to navigate to when the code is scanned.  For example, if using Patient Online Scheduling you could enter the practice site for this.  Or you could enter your general practice website.  The QR Code can be further formatted by color, background, alignment, etc.  This image file can also be copied and pasted into other applications once it is created.
