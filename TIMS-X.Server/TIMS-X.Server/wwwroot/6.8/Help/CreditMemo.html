  1. POS

    Credit Memo

    Please refer to the directions under Invoice on how to manipulate Item Lines.  On Credit Memos, a payment will represent a refund. 



    Select Source of Document

    Click New to create a new Credit Memo.  Click Open to select from a list of the patient's non-finalized credit memos.  Click All to select from a list of all the patient's credit memos (finalized or open).

    Patient

    The name and active address of the selected patient.

    Bill to Customer

    Will default to the patient’s name and active address in the Patient details. If

    an address is entered in Name & Address for Invoicing - this will be the default used.

    All fields can be edited as desired, and will be saved with the invoice only.

    Payment Type

    If a payment is to be entered onto the credit memo, you can choose a type here, which will determine further processing after the credit memo is finalized. The default is Manual.  Payments entered on credit memos will be processed as a refund.

    Document Information


     Document Date

    Date of the credit memo. Will default to the current date, but can be changed as needed.

     Site

    This will default to the site specified in the Patient details, but can be changed as needed.

     Provider

    Name of provider who will be credited for this return.  This is a required field

    and should match the provider of the original invoice that the credit memo applies to. If you are crediting a hearing aid from the HA History link, and that device was previously auto-invoiced, the Provider will default to the same one from the original invoice. This information will be exported to the Sales Rep field in the QuickBooks® credit memo.

     Applied Amount Remaining

    The amount of the credit memo that has yet to be applied to an invoice.

     Tax Rate

    Specifies tax group for the invoice. Will default to the tax group chosen as “Default” in Setup.

    Line Details


     Item

    The sale Item name - either a POS Item, HA Model, Battery Size, or CPT Code

     Description

    Description of the item. Will default to the description entered in setup. This field can be edited as needed.

     P.O. No.

    Not used for credit memos.

     Unit Price

    Price per unit. Will default to the price entered in the Setup. Can be edited as

    needed.

     Units

    Quantity or units of items, can be edited as needed.

     Amount

    Total amount for the given item; automatically calculated from the quantity and the unit price.

     Tax

    If the line item is taxable, this box will be checked. This will default to the taxable status entered in the Setup. Taxable status can be changed by highlighting the line and clicking the Edit button. (or double-clicking on the line) 

     3rd Party

    Not used for credit memos.

    Document Totals

    The subtotal (without tax), tax total, total credit amount, and remaining balance (including other applied payments) of the invoice.

    Office Notes

    Open field for internal notes only.  These will not be printed on the customer's receipt, or exported to QuickBooks®.

    Customer Message 

    Customer message that will print on the invoice. Messages can be added in Setup.

    Memo

    Open field for memo notes. These can be included on a printed document if desired, and will be exported to QuickBooks.

    Line Commands

    • Add - add a new line item to the credit memo
    • Insert - insert a new line item  to the credit memo above the highlighted line
    • Edit - view and/or edit details for the highlighted line item 
    • Remove - deleted the highlighted line item from the credit memo.

    Document Commands

    • Final - lock the credit memo so that it can no longer be edited.  Only finalized credit memos will be exported to QuickBooks.
    • Print - click once to open a preview window.  Click the print icon on the preview to print to printer.   Note: Printing an open POS credit memo will automatically finalize it.
    • Documents - this button will only be active if the credit memo has been finalized.  Click here to run reports for the active patient or credit memo.
    • Delete/Void - If the credit memo is still Open, this button will display "Delete".  If finalized, it will display "Void". Deleting a credit memo will remove it entirely from the database.  This is only allowed with an open credit memo.  A finalized credit memo can only be Voided.  A credit memo that has been exported to QuickBooks cannot be deleted or voided.
    • Apply POS - click this button to apply the credit to a finalized invoice.
    • Close - close the active credit memo, but remain in the POS module.  
    • Exit - close the POS module entirely
