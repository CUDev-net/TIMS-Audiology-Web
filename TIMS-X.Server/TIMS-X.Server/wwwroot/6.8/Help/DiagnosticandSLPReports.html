    Diagnostic and SLP Reports

    Diagnostic reports are generated from the history information, and can include the audiogram and other testing data, along with additional details from the patient and referring physician record.  

    History Diagnostic Reports

    With the desired History session selected on the History Patient Panel, click the Report button  to select a diagnostic report. Choose from the list of available reports that have been added through the report designer, or as custom reports. 

    The report chosen as the default in Practice setup will be at the top of the list. Custom reports can also be used with this feature - contact TIMS Support for details. 


    • The report viewer will display the report.
    • If the report has been setup in the report designer as Editable at launch, you will be able to make changes, type additional notes, remove text or graphics, etc. 
    • If not designated as editable, the report will be displayed as read only.
    • From the report viewer, several actions can be performed from the File menu:

      • Print the report. Depending on the Auto Archive setting chosen in the report designer, either a message will ask if you wish to archive a copy, it will automatically archive, or the window will simply close after printing.
      • Contact TIMS Support for assistance in setting up your practice for electronic faxing and emailing. Then make sure accurate fax numbers have been entered in the Physicians setup.  For emailing to a physician, an email address must be entered into the Secure email field.
      • Email the report (fees applicable - contact TIMS Support at 800-763-0308 for details) The email panel will appear to the right of the report.

        • Check the Physician(s) to email the report to.  (Only physicians with secure email addresses entered will be listed)
        • Enter a One-Time Email address if desired.
        • Check Email Patient to send the report to the patient as well.
        • Edit the Subject and Body of the email if desired.
        • Click Send to send the email, or Cancel to abandon changes.
      • Fax the report (fees applicable - contact TIMS Support at 800-763-0308 for details)
        • The fax panel will appear to the right of the report.

        • Check the Physician(s) to fax the report to. The History Referring Physician, and the patient's Referring and Primary Physicians will be at the top and checked by default. Only physicians with fax numbers entered will be listed.  If the physician has multiple fax numbers, they will be listed under the "Main Fax". Click the down arrow to select an alternate number.

        • You can also send the fax to a one-time number.  Enter it in the One Time Fax # box. A One Time Fax Name can also be added to be included on a coverage page.

        • Choose a Cover Page to be included, if desired. If a cover page is used, Subject must be entered.

        • The From Name will default to the logged in user/provider - this can be changed if needed (only if a Cover Page is selected). Subject is required if a Cover Page is selected. Enter any Comments as needed- they will appear on the cover page if added to the template. The From Phone will default to the logged in Site phone number - this can be changed if needed.
        • On the Cover Page, the To Name will be the Physician selected or the name entered in One Time Fax Name.
        • Click the Preview button to see what the cover page will look like.  Sample information will be displayed for the from and to information, but the subject and comments will show as entered. Click Cancel to close the preview.
        • Click Send Fax to send the fax, or Cancel to abandon. A confirmation message will popup:

        • Click Yes to proceed, or No to cancel.
      • Archive the report. If the report has been setup in the report designer as Editable at launch, a message box will popup 

        • Save Draft - the Save draft option will be available, only if the report has been setup as Editable at launch. Choose this option to save the changes made so far, and return later to finish. If a draft report has been saved, it will be indicated in the report list the next time it is run. 

    When run again the Use Draft? message will give the option to either edit the existing draft, or discard it and start new.

          • See below for details on Tracking Changes in draft reports.  All user changes can be tracked and reviewed before the final archive.
        • Archive to patient reports  - save a snapshot of the report to the Archived Reports folder in Patient Activity.
        • If the report has been not been setup as Editable at launch, then the confirmation message box will simply confirm that you wish to archive.  Click OK to archive, or Cancel to abandon.

    SLP Reports

    If the History Type of the selected History record is an SLP specialty, this screen will automatically be shown by default. Click on the Reports tab and choose from the available SLP reports that have been added through the report designer. Viewing, editing, and archiving of the report is the same as described above for diagnostic reports.

    Tracking Changes in Reports

    If the report has been setup as Editable at launch, all changes made to the document can be tracked and reviewed before it is printed and/or archived.

    • Run an editable report as described above to display the report in the viewer. 
    • Note - each user must click the Track Changes each time before the draft document is edited, and the changes need to be tracked.
    • Click the Proofing tab, then Track Changes.

    • Make changes as needed to the document.  You can tell that changes are being tracked, if the text displays with an underline and a different color font.

    • Close the document and save as a Draft as described above.  When the next user opens the draft, the previous user's changes will be displayed, still underlined.  Click Track Changes again if more changes need to be tracked. If the new user makes changes, theirs will also be underlined in a different color.

    • Comments can be added by highlighting text then clicking the Add Comment button.

    • Click Comments to view all comments that have been added so far. Any user can reply to previous comments, edit them, or delete.

    • To see all changes that have been tracked so far, click Tracked Changes. Individual changes can be Accepted or Rejected here, or all of them can be Accepted or Rejected.
    • When all changes have been either accepted or rejected, the document can be printed, emailed, faxed, and/or archived as described above. Note: If changes are not reviewed and either accepted or rejected, they will still appear underlined and in different font color in the final document.
