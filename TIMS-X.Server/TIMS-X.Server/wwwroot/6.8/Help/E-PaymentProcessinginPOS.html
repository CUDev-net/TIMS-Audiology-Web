    E-Payment Processing in POS

    E-Payments can be processed in the following POS areas:  Invoices and Payments.  The functionality of E-Payment processing is largely the same for both areas.


    Existing TIMS users should already be familiar with POS Payment Items.  In the past, these payment items were simply a way of representing “manual” payments made outside of TIMS (i.e. actual payments made with third-party software or payment terminals).


    Offices can continue to use the POS Payment items in this fashion, but can now also choose to process the payments directly through TIMS.  

    • Manual - Use this selection to retain traditional payment functionality in TIMS (i.e. if an office elects to continue to process E-payments outside of TIMS).
    • Credit Card - Select this option if the office intends to process a credit or debit card through TIMS.
    • E-check - Select this option if the office intends to process a check payment through TIMS.
    • CareCredit - select this option to process the payment through the CareCredit™ system.


    Steps for Manual:

    1. Enter line items, including any relevant Payment Items (Payment Method and Reference are required).
    2. Click Finalize Document , then Yes to confirm.
    3. Confirm Manual Payment type and amount, then click OK.


    Steps for Credit Card

    1. Enter line items, including the relevant Payment Items, and then select Credit Card option.
    2. Click Finalize Document , then Yes to confirm.
    3. The E-Payment Processing window will popup.

      1. Select Manual Transmit to process the e-payments outside of TIMS.
    1. Swipe Card, then confirm card number. 

    1. Click Card Transmit (an Approved/Denied window will then pop up)


    Notes:

    • “Please Swipe Card” must be flashing before the card is swiped.  If this message is missing, click the Re-Swipe button before card is swiped.
    • If a POS Printer has been setup properly (i.e. using a valid local or network printer), two receipts will be automatically printed following the Approved/Denied window. 
    • If no printer was setup (or setup incorrectly) a traditional Windows print screen will pop up asking the user to direct the printing to a valid source (a printer, Microsoft XPS Document Writer, etc.).
    • A similar process applies to POS Payment receipts.  Select the Credit Card or E-Check option and continue on with the Payment, which will eventually lead you to the same Credit-Card or Electronic Check Processing windows that are described above.


    Steps for E-Check

    1. Enter line items, including the relevant Payment Items, and then select E-Check option.
    2. Click Save.
    3. Click Finalize Document, then Yes to confirm.
    4. Swipe check, and confirm Bank, Account, and Check Number. 

    1. Click Transmit (an Approved/Denied window will then pop up). 

    Steps for CareCredit

    • See CareCredit™ Financing

    Refunds

    TIMS E-Payment Processing also facilitates refunds.   POS refunds for Credit Cards or E-Checks can be issued through the POS Payment window.  In order to process a Credit, Debit, or E-Check refund, please note the following requirements for a standard account setup with TransNational:


    • Refunds must be made against existing credit card or E-check payments
    • Refunds cannot exceed the amount of previous payments.


    Users should also note that they do not need the credit card or check information to issue a refund.  TIMS has a transaction history dropdown (titled Trans ID Lookup) that will provide users with the information they need to process refunds.


    In addition to the standard setup above, customers have the option to enable “Pure Credits” with TransNational.  A Pure Credit is the ability to issue a stand-alone credit to any credit card the customer wishes, regardless of any past or related TIMS POS transaction information.  This feature is not enabled by default on TransNational accounts.

    POS Payment (Payment Reversal) - Standard Account

    To issue refunds (credits) through the POS Payment screen,  these requirements apply:   refunds must be made against existing credit card or E-check payments, and refunds cannot exceed the amount of previous payments.


    1. Under Payment Options, select the Credit Card or E-Check option.
    2. Enter Payment Amount (the amount to be refunded, which is always positive).
    3. Check the Payment Reversal box.
    4. Click Save, then Finalize Document, then Yes twice to confirm.
    5. Click NO on the Credit Confirmation window, which automatically selects the Transaction-ID Credit, which is the credit  type required for Standard Accounts.
    6. Use the Trans ID Lookup dropdown menu to identify a previous Transaction ID to apply the Credit to.
    7. Click Transmit (an Approved/Denied window will then pop up)


    If a customer chooses to enable the “Pure Credit” feature with TransNational, then the following procedure applies when entering Pure Credits:

    1. Choose Payment Option (ex. Credit Card).
    2. Enter Payment Amount, and check Payment Reversal.
    3. Click Save, then Finalize Document, and click YES twice to continue.
    4. Click YES on the Credit Confirmation window,, which automatically selects the PURE Credit option.  This credit will not be tied to any previous POS transactions.  It stands alone.
    5. Enter the appropriate Card information.
    6. Click Transmit (an Approved/Denied window will then pop up)
