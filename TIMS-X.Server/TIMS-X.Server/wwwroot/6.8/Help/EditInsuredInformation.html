    Edit Insured Information

    Primary

    To edit the Primary Insured information, click the Insurance Payers button .  The Insurance Editor screen will popup.  

    If an insurance payer has already been billed for the claim, it cannot be changed.  However, all other insured information can be edited. After editing the information, click OK to save, or Cancel to abandon your changes. Any changes made to the insured information will be reflected in the Patient details, this claim, future claims (including resubmits), and/or a printed 1500 form.


    Secondary

    If there is a secondary payer, you can edit the Secondary Insured information by clicking the Insurance Payers button.  If there is no secondary payer, you can also add one here by clicking the same button.  The Insurance Editor screen will popup.  For an edit, all the existing insured information can be changed, but the Carrier Name can only be changed if it has not already been billed for the claim.  To add a new secondary payer, choose the Carrier, then the Patient Relationship. If you choose "Self" as the relationship, the address birth date, sex, and employment information will automatically be copied from the patient's record.  (this can be changed if needed). After editing the information, click OK to save, or Cancel to abandon your changes.  Any changes made to the insured information will be reflected in the Patient details, this claim, future claims (including resubmits), and/or a printed 1500 form.


    Additional Payers

    If there are additional payers, you can edit the Insured information by clicking the Insurance Payers button.  If there is no tertiary, quaternary, or quinary payers, you can also add them here by clicking the same button. The Insurance Editor screen will popup.  For an edit, all the existing insured information can be changed,  but the Carrier Name can only be changed if it has not already been billed for the claim.  To add a new payer, choose the Carrier, then the Patient Relationship. If you choose "Self" as the relationship, the address birth date, sex, and employment information will automatically be copied from the patient's record.  (this can be changed if needed).  After editing the information, click OK to save, or Cancel to abandon your changes.  Any changes made to the insured information will be reflected in the Patient details, this claim, future claims (including resubmits), and/or a printed 1500 form.


    Claim

    Click Edit button to edit the selected claim information.

    The Claim Editor screen will popup.  

    There are 5 fields that can be edited or entered here and will carry forward if the claim is resubmitted.

    • Prior Authorization Number  
    • Box 19 (Reserved for Local Use) on a printed 1500 form
    • Claim Frequency Code
    • Payer Claim Control Number 
    • Referring Provider - can also be copied to the patient information if desired, by checking the Update Patient Information box.


    The Bill To Provider and Supervising Provider (if used) will be displayed, but are not editable. Tracking information for the claim (which user and site created and last updated) will also be displayed.
