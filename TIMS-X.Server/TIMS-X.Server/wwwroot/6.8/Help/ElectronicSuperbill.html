    Electronic Superbill

    Patient charges for each appointment can now be entered in the History link, then later accessed from the Billing Center.  Services and items that will be billed to either insurance (claims) or the patient (POS invoice) can be included on one superbill document. When all charges have been entered, the superbill is finalized to automatically create a claim transaction and/or invoice. 


    The Electronic Superbill is tied to a date of service via a regularly scheduled appointment.  To create an electronic superbill for a patient, the practice must utilize the "Create History from Appointment" feature in Practice Setup, and the patient  must have an appointment that has a start time no more than 30 minutes in the future. 

    Adding Lines to the Superbill

    • Highlight the desired History record in the patient tree, so that the details are displayed on the right.  Click the Superbill button to display the patient's new superbill for that visit.

                 

    Note: The Superbill button will not be active if the selected History record is not linked to an appointment.

    • Add a POS Line
      • Click the Add POS Line button to add a POS line item for that specific visit. The new POS line window will pop up.

      • Click the ellipsis button  to display the Item Lookup window where you select the POS item you want to add to the superbill.  Select the POS category and choose the POS Item from the list by double-clicking on the item. This will populate the Superbill Item window with your selection and fill in the default price and quantity.  Both the price and quantity can be adjusted accordingly for this specific visit.  Add any applicable notes about that POS Line and click OK to save the line or Cancel to abandon changes.. Upon saving the line it will be displayed in the Superbill window where you can double-click the line to edit or single-click to select the line for deletion using the Delete Line button .  Click the Add POS Line button to add another line.
    • Add a Claims Line
      • Click the Add Claim Line button to add a Claim Transaction line. The new Claim line window will pop up.

      • Select the CPT Code, Diagnosis Code(s) and Modifier(s) (if needed) for the services performed.  If necessary, the price and quantity for this specific visit can be modified using the related fields.  You may also add any notes associated with this particular charge line and they will be visible in the Claims - Transactions link.  Note that at least one Diagnosis Code is required to save a claim line.  Click OK to save the line or Cancel to abandon changes.  Upon saving the line it will be displayed in the Superbill window where you can double-click the line to edit or single-click to select the line for deletion using the Delete Line button .  Click the Add Claim Line button to add another line.  
    • Add Multiple Claim Lines at Once
      • Click the Add Claim Group button .  The Superbill multi-item selection window will popup. 

      • The top 15 most used CPT and Diagnosis codes will be at the top of the lists, with an asterisk (*) in front of the name.  Simply click the codes to be used for the visit.  At least 1, and no more than 4 Diagnosis codes may be selected.  Click a line again to un-select it.   Click OK to save the claim group or Cancel to abandon changes. The lines will be added to the superbill, along with the corresponding diagnosis code(s).
    • As you add lines, the total charges will be reflected on the bottom of the superbill.

    Auto-Invoicing

    Automated Invoicing functions much the same way in the Superbill as it does in POS. When a new superbill is created for a patient and there are open items in the queue to be invoiced, a confirmation box will pop up. 

    Click Yes to add the item to the superbill, or No to ignore.  If the item is added as a superbill item, it will no longer be in the billing queue and you will not be prompted again when creating future POS invoices. Items that are available for auto-invoicing are: 


    • Hearing Aid Histories that have the “Invoice” box checked
    • Repairs marked as “Payable”
    • Battery Club Joins and Add to Balance actions.

    Finalizing the Superbill

    When all applicable POS and Claims item for that specific visit have been added to the Superbill, click the Save button . Once the Superbill has been saved, the Delete and Finalize buttons will be activated.  Delete will delete the Superbill and all associated lines. (Any auto-invoiced items will be put back into the billing queue.)


    Finalizing the Superbill will create a POS Invoice for all POS lines and/or a Claims Transaction for all claims lines.  This action will also prompt TIMS to look for any additional items that need to be auto-invoiced and prompt you to add them to the Invoice.

    Note: The Superbill does NOT have to be finalized in the History window to allow you to add Hearing Aids, Battery Club or Repair items.  However, once a Superbill is finalized, you cannot add additional lines.


    If POS Lines exist on the Superbill upon finalization, the Invoice button at the bottom of the Superbill will be activated where the user can click to view the POS Invoice automatically created by TIMS with the corresponding POS Items from the Superbill.


    Treat this POS Invoice as any other and add additional lines (charges and/or payments) as necessary.


    Any claim lines on the Superbill will be shown in the Transactions link for that patient on that visit date where additional transaction lines and/or details can be added as necessary.


    Details of all finalized superbills, and the linked POS invoices and Claim Transactions can be viewed in the audit report Superbills Posted by Date. Under each superbill will be listed details of claim transactions and POS invoices, both created from the superbill, and also linked to the same appointment.  The purpose of the report is to display all billed items associated with the same appointment and compare what was entered on the superbill with what was actually billed.  User information shown when available. It can be used to see if items entered on the superbill were deleted or changed, or any items were double-billed.  Deleted lines or invoices will be shown in red.  Optional parameters for site, and patient last name.  This report can be found under the menu TIMS Reports - Management. 

    Billing Center

    Click the Billing Center icon to display a list any Open Superbills or Open POS documents for the selected Date Range.  Double-click and open Superbills or POS documents to view them.
