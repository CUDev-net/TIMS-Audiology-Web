    Generate Letters 

    1. Important: In order to build a Patient Letter List, you must first select a letter from the Letters panel. 

    1. Choose one or more criteria, as described in Creating the Patient List. Click the Build List button to generate the patient list.  
    2. A message will popup asking if you wish to save the criteria/parameters used.  

    If you frequently use a certain set of criteria for a particular letter list, this can be very helpful.  Once the parameters are saved, the next time the letter is selected the same criteria will be loaded.  Previously saved parameters can be cleared by clicking the Reset Stored Parameters icon on the Letters Panel. 

    1. Depending on the merge fields that you included in your letter, you may be prompted to select certain criteria, if you haven't already done so before clicking Build List.  For example, if your letter contains fields from the Appointment Source you must select at least one item from the Appointment Filter Criteria section.  If you don't, you'll get an Invalid Criteria message, and the appropriate filter criteria section will open and be highlighted.

    1. The resulting patient list will display the Name, primary Phone, default Site, and Zip Code of the patients who satisfy the selected criteria.

    The Email box will be automatically checked if the patient has an email address already entered.  The Print box will be automatically checked unless: 1) the patient is marked as No Mail in Communication Restrictions or 2) the patient does not have a complete address entered. The patient name will be displayed in red font if he/she has any Communication Restrictions. Holding the mouse cursor over the name will popup a tooltip listing the restrictions.

    1. Click on a name in the list to see a preview of the letter in the Letter Preview tab. You can check to see if there is any missing or incorrect patient information. If you notice anything that is not right, you can click on the adjacent Patient Activity or Summary Panel to access the patient's information and correct it. When you re-select the patient's name for preview, the new information will be reflected.

    1. Un-select any names you do not wish to process (Email or Print) by un-checking the box under the appropriate column.  You can un-select all in a column by using the icons at the top of the panel.  If you need to exit Communications and come back later, you can save the list with the current selections.  Click the Save checked patients button, and the list will be the same when you return.

    1. When you have the list in place that you wish to print/email, click the Process Letters icon .   Depending on the letter's setting for Auto Archive, a message may popup asking if you wish to archive a copy of the letter for each patient.  

    Click Yes, No, or Cancel to stop the processing.  

    1. Next, a message will popup confirming that you wish to process the selected number of letters.  

    Click Yes to continue.  Select your printer then click OK.  

    1. A progress bar will show while the letters are printing.  

    1. A confirmation message will let you know when they are done.

    1. If you have chosen to archive the letter in the patient's record, 2 entries will appear in the Patient Activity Panel - one under Archived Letters, and another under Communication Interactions.

    1. You can view, re-print, email or fax a copy of the archived letter at any time by double-clicking on it under Archived Letters. Any re-prints will be recorded in the Print History list to the right of the letter.

    Mailing Labels

    1. To print mailing labels, first select your criteria in the same way as with a letter.  Then select a label template from the Mailing Label category in the Letters Panel. 

    1. A standard template for 3 x 10 labels (for US) is automatically included in TIMS. For Australia, New Zealand and UK 3 x 7 labels are included.  
    2. After selecting the label template, click the Build List button to generate the patient list.  
    3. Unselect names as needed as described above.  When you are ready to preview the labels, click the Print Mailing Labels icon .  
    4. A preview of the labels will be shown in the Letter Preview tab.

     

    1. To print the labels, click the File tab, then Print. The print window will popup. Print the labels to the desired printer. Printed mailing labels are not archived.
