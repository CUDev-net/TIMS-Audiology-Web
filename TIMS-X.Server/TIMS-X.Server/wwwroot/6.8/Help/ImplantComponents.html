    Implant Components

    The Implant Component section contains the repair information for the selected hearing aid.



    • Implant Component information can be hidden or displayed by clicking the arrow.
    • "Implant Components" heading appears in black when no components have been entered.
    • "Implant Components" heading appears in orange when at least one component has been entered.
    • Implant Component information displayed includes the component Type and Manufacturer/Model, the date it was received by the patient, and it's current status. 

    Implant Component Icons

    • Add Component
    • Edit Component
    • Copy Component to other side (binaural HA History only)
    •  Delete Component
    •  Attach Document to Component (viewed in the Imaging Center)


    Implant Details

    Component Type

    The type of component added with the implant. Component types can be added or edited in Setup. 

    Manufacturer/Model

    A free form field to enter the Manufacturer and/or Model name of the component.

    Status

    The status of the component.  Available choices are those added in Setup.

    Date Received

    The date the component was received by the patient.

    Date Warranty

    The date the warranty on the component expires.  

    Date Returned

    The date the component was returned, if applicable.

    Serial Number

    A text field for entering the serial number.

    UDI Number

    A text field for entering the UDI number.

    Notes

    A free form field for miscellaneous notes about the repair.  Up to 1000 characters of notes can be entered to record any special circumstances.

    Tracking


      Updated - By User/By Site

    The date when the component information was last updated, along with the User and Site (location). This is automatically updated when the record has been changed and saved.

      Created  - By User

    The date when the component information was originally created, along with the User who created it.
