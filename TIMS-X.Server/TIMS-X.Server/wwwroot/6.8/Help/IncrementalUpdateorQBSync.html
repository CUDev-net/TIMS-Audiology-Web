    Incremental Update or QB Sync

    The incremental update (QB sync) is run to export Claim or POS Invoices, Credit Memos, and Insurance Transfers, POS or Claim Payments, Claim adjustments, add new patients or items, and update patients or items in QuickBooks®. The boxes checked in QuickBooks Setup will determine which invoices and payments/adjustments will be exported. A posting report is generated every time the update is run. The update can be run at any time and there is no limit to the number of times it can be run.


    To run the update:

    1. Click on the QuickBooks link .

    1. Click the Incremental Sync button.

    1. The QB Sync screen will popup.  Click the Start button.

    1. This process will export any new or changed information pertaining to the given export lines found in your TIMS database since the last time this procedure was executed. Invoice, payment, adjustment, and POS exports will depend on which categories have been previously checked in the QuickBooks Setup.
    2. As the export runs, a check mark will display next to each exported line.
    3. When the procedure has completed the “Finished” button will be active. Click on Finished.
    4. A message will popup asking if you wish to run the QBSync Report.  

    The QuickBooks Posting Report will give you the details of all exported items for that sync. Be sure to check the non-exported items and invoices at the beginning of the report and call TIMS support if you have questions on how to fix them.  


    Note: If you close the report, you can always view it later under Reporting, QuickBooks, QuickBooks Posting Report.
