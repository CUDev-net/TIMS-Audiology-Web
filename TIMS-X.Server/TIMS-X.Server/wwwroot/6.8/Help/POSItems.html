    POS Items

    The POS Items that are not in your other lists: Hearing Aid Models, Batteries and CPT Codes. There are 7 types of POS Items: 

    • Comment - use to type notes directly in the Description field of the item line. There is no price for a comment item. If you have standard notes that you use frequently, you can create several comment items with that note in the Description field. Otherwise, leave the Description blank and type your note as needed when used in an invoice or credit memo.
    • Discount - use to subtract a percentage or fixed amount from the line item directly proceeding it.
    • Inventory - use for items that are purchased, tracked as inventory, and resold.
    • Non-Inventory - use for items that are purchased and resold, but not tracked for inventory purposes.
    • Other Charge - use for miscellaneous charges, such as shipping, service charges, etc.
    • Payment - use to record a payment received at the time of the sale. It reduces the amount owed on an invoice. A payment type item used on a credit memo represents a refund.
    • Service - use for services not included in CPT Codes (not billed to insurance). The service item "Repair" is protected, and cannot be renamed or deleted.


    No duplicate names are allowed in POS Items - an item cannot have the same name as another POS item, CPT code, battery size, or hearing aid model. An item cannot be saved without an Item Type selected.


    To add a new POS Item, select the tab of the Type you want to add, and then click the Add New button. 


    Name

    The name of the POS Item, that will appear in the Item drop down list of the invoice or credit memo.

    Description

    The default description of the POS Item that will appear in the 'Description' field of the invoice or credit memo. It can be edited directly on the document.

    Price

    The default price that will appear in the Unit Price field of the invoice or credit memo. It can be edited directly on the document.

    Taxable

    If Checked, sales tax will be applied by default when the item used in an invoice or credit memo. The taxable status can be changed directly on the document.

    QB Account

    If a special GL sales account is to be used when this item is invoiced and exported to QuickBooks, select it here.  Otherwise, leave as "Default". This is not available for Payment or Comment POS item types.

    Payment Method

    If the item is a Payment type, it must be assigned to an existing Payment Method. 

       

    Is Percent

    This field is only active if the type is Discount. If checked, the price is subtracted as a percentage of the preceding line item. If unchecked, the price is subtracted as a flat amount.

    Inactive

    Check this box if the item will no longer be used.

    Is Repair

    For Service type items, check this box to designate the service as a Repair type. This will make the item available to be auto-invoiced for payable repairs.
