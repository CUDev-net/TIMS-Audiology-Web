    POS Tax

    Note:  Taxes must be setup in this order: Tax Agencies first, then Tax Items, and finally, Tax Groups.

    POS Tax Agencies

    The agency that a tax item is paid to. Each tax item must have an associated tax agency, so you must add the agencies first, and then the tax items. No duplicate names are allowed in Tax Agencies - an agency cannot have the same name as another agency, tax item, POS item, CPT code, battery size, or hearing aid model.


    Agency Name

    The name of the tax agency.

    Description

    The description of the tax agency.

    Inactive

    Check this box if the item will no longer be used.

    Tax Item

    The individual tax items that calculate a single sales tax at a specific rate and are paid to a single tax agency. Each tax item must have an associated Tax Agency. To add a new Tax Item select desired Tax Agency first, and then click the Add new button.  One or more tax items are combined into a Tax Group that displays a total rate on the invoice or credit memo.


    Name

    The name of the tax item.

    Description

    The description of the tax agency.

    Rate %

    The rate designated to that tax item. Rates can be entered up to 4 decimal places.

    Inactive

    Check this box if the item will no longer be used.

    Tax Groups

    Tax Groups are used to combine one or more tax rates into a total rate that is shown on the invoice or credit memo. For example, you may have a total tax rate of 5%, which represents 3% state tax, 1% county tax and 1% city tax. 

    Note: Only tax groups are shown in the drop down boxes in the document, so even if you have only one tax rate, it must be included in a tax group. You may setup as many different tax groups as you need, if you have different tax situations at different locations or for certain items.  To add a new tax group, go to the POS Tax Groups tree, and then click the Add new button.


    Group Name

    The name of the tax group.

    Description

    The description of the tax group.

    Inactive

    Check this box if the item will no longer be used.

    Tax Items

    All individual tax items will appear on this list. Check the boxes of the items that are to be included in the group, to get the desired total tax rate.
