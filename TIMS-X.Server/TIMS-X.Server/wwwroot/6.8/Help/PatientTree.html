    Patient Tree

    Patient Tree Icons

    • Add New Patient
    • Add New Appointment for selected Patient
    • Show All Appointments for selected Patient
    • Refresh Transactions list


    • The Patient Tree displays the names of the 20 most recently accessed patients. 
    • Recent patient names are stored per user so one user’s access will not change another user’s tree.
    • An arrow symbol to the left of the name indicates that the patient has a past appointment. By default, only the patient's appointments from the last year are displayed. To see all of their appointments, click the Show All Appointments icon at the top right of the tree.  Click the arrow to expand the tree and view the appointment date and time. 

     Any appointments with a status set not to show in the appointment grid will not be displayed.  

      • Appointments that have had no transactions added yet will be in regular font (not bold).
      • Appointments that have had any transactions that have been generated into claims, will be in bold orange.
      • Appointments that have had transactions added (but not yet generated into claims) will be in bold black.
      • The gender symbol next to the name indicates whether the patient is male or female. A name in gray font indicates an inactive patient.


    Select an appointment and the details will appear on the screen to the right.  Double-click on the date to edit the appointment details.

    Patient Search

    • The Patient Search function allows a user to search the entire TIMS database for a patient that is not currently on the patient tree.

    • At least two letters or numbers must be entered in order to enable the Magnifying Glass search icon and list the search criteria options.
    • Searches can be made for a patient based on Last Name, First Name, Birth Date, SSN, Phone Number, Account Number, Address Line1, City, Zip Code, HA Serial number, PO number, or Repair number.
    • There is a check box to include Inactive patients.
    • Including Inactive patients is useful to ensure duplicate patients are not created.
    • Click on the Magnifying Glass search icon or press the Enter key to initiate the search.
    • To exit the Search screen, clear search box of all entered letters or numbers.

    Add a New Patient

    If the patient you are searching for is not in the database, you can add the patient by clicking the Add New Patient icon .


    The Add New Patient screen will popup.  Enter as much information as possible, to save time later.  The fields highlighted in yellow are required, and are customizable in Practice setup. A Marketing Source may also be required, depending on the practice rules. After entering all available information, click OK.  The new patient's name will now appear at the top of the Patient Tree.

    Add a New Appointment

    If you need to create a new appointment for the patient, highlight the name, and click the Add Appointment icon.  


    The Appointment Details pop-up screen will appear, allowing you to enter the appropriate details.  

    Provider will default to the patient's default provider, if one has been pre-selected in Patient details. Site will be pre-selected to the patient's selected default site. Both can be changed as needed. Click Save, or Cancel to abandon changes.  See Appointment Details for more details.
