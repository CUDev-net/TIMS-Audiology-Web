    Physicians

    The name of the provider who referred the patient to your practice, or is the patient's primary care provider. Physicians can also be entered in the Marketing Sources setup, under the category "Physician Referral".  Any physicians entered in Physician setup can be viewed in Marketing-Physician Referral setup and vice versa. Changes made in either location will also be reflected in the other one.


    Salutation

    The salutation of the physician, if needed.

    First - Initial - Last

    Name of the physician.

    Phone - Ext

    The telephone number, including the area code and extension (if needed), of the physician.

    Practice

    The physician practice name, if needed.

    Address1 - Zip Code

    The mailing address of the physician. Click on the Verify Address button to lookup and confirm the official address.

    Email

    The email address of the physician.

    Secure Email

    The secure email address of the physician. To use emailing of diagnostic reports, this field must be filled in.

    Notes

    Free form field to put in notes about the manufacturer.  Up to 255 characters can

    be entered.

    Primary Fax

    The main fax number for the physician.  To use electronic faxing of diagnostic reports, this field must be filled in.

    Other Faxes

    If the physician uses any other fax numbers, they can be entered here.  Click the button to add additional numbers. Select a number and click the up/down arrows to change its priority.

    Tax ID

    The Tax ID used by the physician.

    Tax ID Type

    Either Employer Tax ID or Social Security number.

    NPI

    The National Practitioner Identifier number - a unique identifier for health care

    providers.   To use the NPI lookup tool, click on the magnifying glass icon . The NPI Lookup Form will popup. The First, Last, City, State and Zip Code of the physician will be pre-populated if that information has already been entered.  Change the search fields as needed and click Search. Double-click on the desired match to enter the NPI found.      

    Secondary ID

    A supplemental ID assigned to the physician, for insurance purposes.

    Business Agreement

    The date a privacy agreement was signed with this physician.

    Next Contact

    The date the practice wishes to contact this physician next for further outreach/communication.

    Sites Where Available

    Check the specific sites where this physician's name will available as a selection. Or check "All Visible Sites" for all. At least one site must be selected.

    Inactive

    Check this box if the physician will no longer be used.
