    Practice

    The Practice setup is where the basic and important information about the entire company is entered, including address, relevant ID numbers, and certain global settings.  Only 1 Practice can be entered.  A Practice can not be deleted or marked as inactive.


    Practice Name

    The name of the pay-to provider practice. Up to 100 characters can be entered, although only the first 33 characters will be sent with electronic claims.

    Name - First, Initial, Last

    The Name of the pay-to provider. Name must be entered if SS# is used as the Tax ID.  (enter it only if this is the case)  

    Fax Number

    The fax number for the practice.

    Email

    The email address for the practice.

    Online Scheduling Message

    This button will only be visible if Online Patient Scheduling has been enabled for the practice. Click to add any additional text you would like to include on the confirmation message patients receive after they submit their request. This message will both appear on the screen, and in an email sent to the address entered. The bold text at the top is automatic. Also available to view or copy will be the URL to use for patient's to access online scheduling.

    Web Office Code

    This number will be provided by support if the office is using web appointments. It is not editable in this screen.

    Locale

    This regional code to determine address conventions and which paper size and date format is used in reporting.  Enter either US, CA (Canada), AU (Australia), NZ (New Zealand), or UK (United Kingdom).

    Claims Site ID

    A code assigned by the submitter used to identify the provider's specific pay-to

    location, for electronic claims billing purposes only.

    Tax ID Type

    Either Employer Tax ID or Social Security number. If SS# is used, First and Last Name must be entered.

    Tax ID

    The Tax ID number of the practice.

    Billing Address


      Address1 - Zip Code

    The main billing and correspondence address of the practice.  This is the address that will appear on most reports (unless "Use Site Address for Reports" is checked - see below) It is also the address used for Billing Provider Info (box 33) on claim forms. Click on the Verify Address button to lookup and confirm the official address.

      Phone

    The main billing and correspondence phone number of the practice.

    Pay-To Address


      Address1 - Zip Code

    Enter only if different from the billing address. This address is sent with electronic claims and used in a few selected reports.

      Phone

    Enter only if different from the billing phone number.

    NPI

    The National Practitioner Identifier number - a unique identifier for health care

    providers.   To use the NPI lookup tool, click on the magnifying glass icon . The NPI Lookup Form will popup. The First, Last, City, State and Zip Code of the physician will be pre-populated if that information has already been entered.  Change the search fields as needed and click Search. Double-click on the desired match to enter the NPI found.    

    Taxonomy

    Required by some payers for electronic claims.  It is included on printed claims.

    Fiscal Start Month

    Choose the month that the practice uses for the start of the accounting fiscal year.  This is required for KPI goal tracking.

    Inactivity Timeout

    Enter the number of minutes before TIMS will prompt the user to logout due to inactivity.  If the user does not respond, TIMS will automatically close.

    Use Site address for Reports

    Check this box if you want the individual site address to appear on various reports, such as Intake sheets, instead of the main practice address.

    Accept Assignment?

    An indication of whether the provider agrees to accept the charge determination of Medicare as the full charge.

    Use Site address for POS

    Check this box if you want the individual site address to appear on printed invoices, credit memos and payment receipts instead of the main practice address.

    Use AD Authentication

    Check this box to use Active Directory Authentication, which will allow for logging

    into TIMS with Windows credentials.

    AHAA ID

    If the Practice is an AHAA member, enter the membership number here by clicking on the ellipsis button .   It will be used for certain export functions.

    ESCO ID

    If the Practice is an ESCO member, enter the membership number here by clicking on the ellipsis button .   It will be used for certain export functions.

    Default History Rpt

    The default diagnostic report used in the History link.  This report will appear

    at the top of the list when the Report button is clicked.

       

    Intake Sheet

    New version 6.08. The default report to be used when the Intake Sheet button is clicked in the Appointment Summary panel. This can be either a Selected Appointment Report or a Communications letter/report.  It must contain the word "Intake" somewhere in the name to appear on this list.

    Password Policy

    Click this button to set the desired level of security for user passwords, such as

    required characters, minimum length, and expiration cycle.

       

    Setup Eligibility/Claim Status

    Click to send a request to register for electronic insurance eligibility.

    MFA Code Valid Duration

    New version 6.08. For added system security, practices can now require Multi-Factor Authentication (MFA) when users log in. To turn "on" MFA, enter a value in this field. The value refers to the number of hours between a user being asked for a new code when signing in. In the example below, users would be required to enter an MFA code every hour.  For once a day validation, enter 24. (fees applicable)

    Patient Required Fields

    New version 6.07, you can designate which fields should be required in the patient details.  These fields will be highlighted in yellow, and a patient cannot be saved until the required fields are filled out. Depending on the office setup (contact TIMS Support for details), the chosen fields can be set to not be required on the initial patient add only. When any subsequent patient edits are made, the required fields must be entered in order to save.

    Payer Numbers

    Additional payer ID numbers for specific payer types.  These IDs are generally not used for electronic claims any longer, but may be used in some special circumstances or for paper claims (1500 form).

      UPIN #

    The number assigned to the pay-to provider by the National Registry for Medicare

    Identification purposes.

      State License #

    The number assigned to the pay-to provider.

      Specialty License #

    The number assigned to the pay-to provider.

      Commercial #

    The number assigned to the pay-to provider by the Commercial payer for Identification purposes.

      Medicaid #

    A number associated with the pay-to provider that is assigned by a Medicaid State Agency for identification purposes.

      CHAMPUS #

    The number assigned to the pay-to provider by the CHAMPUS payer for Identification purposes.

      Blue Shield #

    The number assigned to the pay-to provider that is assigned by the Blue Cross/Blue Shield for Identification purposes.

      Medicare #

    The number assigned with the pay-to provider that is assigned by the Medicare intermediary for identification purposes.

      Specialty Code

    The specialty type of practice. Click on the down-arrow and choose the specialty

    from the list.

    Email Disclaimer

    Text that will be displayed automatically at the beginning of any email sent through the TIMS email service.

    Tracking Threshold

    Displays the hearing aid tracking threshold (in months) that has been previously

    setup for Opportunity Tracking.  See Opportunity Tracking Setup for details.
