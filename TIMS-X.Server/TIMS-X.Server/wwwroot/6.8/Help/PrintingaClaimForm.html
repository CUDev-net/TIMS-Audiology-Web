    Printing a Claim Form

    You can print a 1500 paper claim form, on either your own pre-printed forms or on plain paper.  Claims can be printed to request payment from any of the payers on the claim.  To print a claim form, follow these steps:


    1. Click the 1500 or UB04 button in the Printing section.  Note: The UB04 button will only be visible if your practice is set up to process Institutional type claims.

    1. The 1500/UB04 Print Settings screen will popup. 

    1. Choose the type of printing:
      1. Pre-Printed Form- If you are using your own pre-printed forms, check the Preprinted Form box.  If you have not printed before, you may need to print a few experimental copies to get the alignment correct.  
      2. Plain Paper - do not click the Preprinted Form box.  TIMS will print the red 1500 background on plain paper along with the claim information.
    2. Click Print Duplex if you are using plain paper and want to print the instructions on the back side of the 1500 form.
    3. Insured - choose which payer is to be billed to on the 1500.  This will be the payer and insured information that appears on the top of the form, and in Box 11 a-c.
    4. Other Insured - choose which payer will be considered the Other Insured Payer.  This will be the payer and insured information that appears in Box 9 a-d. 
    5. Third Line (UB-04/Institutional only) - choose which payer will be considered the third (line C). 
    6. Top and Left Margin - If you're using a pre-printed form and you know the adjustments needed for the Top and Left margins, enter them now.  Otherwise leave these fields blank until you've printed a test copy.
    7. 1500 Type - will default to the newer 1202 version of the 1500.  (This option is not available for UB-04/Institutional only printing.)
    8. Box 33b - this will display the default value based on the payer type and practice/provider setup.  If the payer requires a different type and ID number to print in this box, you can enter it here.  The values chosen will save with the claim, so that if a resubmit is needed you will not have to enter them again (unless they need to be changed). 
      1. If this value has been changed from the default and the claim from is printed again, you will see an additional button on the print settings screen.  Clicking the "Set to Default" button will reset the Box 33b value back to the default value.

    1. Box 24j (top) - same as 33b above, this will display the default value (usually the provider taxonomy code). If a different value is required by the payer, it can be entered here.
    2. Click OK to print.
    3. A note will automatically be added to the Payer Notes section documenting the date and details of the printed claim. 
    5. If the claim you are printing has been rejected from the clearinghouse, a message will popup asking if you wish to change the status from "Rejected" to "Accepted - Paper". Click Yes to change status, or No to have claim remain as rejected.
