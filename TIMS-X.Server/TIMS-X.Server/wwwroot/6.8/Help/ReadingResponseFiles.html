    Reading Response Files

    1. Once the claims transmit process has finished, a response file (or files) will be downloaded from the Clearinghouse. When a file is retrieved, the following message will appear on the screen. Click OK.

    1. To review responses that have been received, click the ClearingHouse button . The claim responses window will appear.  

    1. Responses displayed will default to the previous 30 days, but can be changed by editing the From and To dates and clicking the Filter by Date button.

    1. The Response Files  list will show the date and time the file was received, along with a response file ID and size.  

    Next to the file date and time will be the type of responses that were received.  Clearinghouse and Payer Errors will be highlighted in red.

    1. When a file is selected, the applicable tabs will be highlighted below. Choose one of the response file tabs to change the view to the report you wish to view.

    Response File Types

    • Ack:  This display shows the acknowledged batch information

    • Status: This display will show the batch status as Accepted or Rejected. (Legacy clearinghouse reports only)
    • 999: This displays information from the 999 acknowledgment file. (Legacy clearinghouse reports only)
    • ClearingHouse Status: This will display the status of the claims in the batch from the clearinghouse. (Accepted or Rejected) This status determines whether the claim is marked as Accepted or Rejected in Adjudication.

    • ClearingHouse Errors: This displays the specific error message(s) for claims rejected by the clearinghouse. The error messages for each claim listed will also be displayed in the individual claim's Clearinghouse Notes section in Adjudication.

    • Payer Status: This will display the status of the claims at the payer. (Accepted or Rejected) This status determines whether the claim is marked as a Payer Reject or not in Adjudication.

    • Payer Errors: This displays the specific error message(s) for claims rejected by the payer. The messages for each claim listed will also be displayed in the individual claim's Payer Notes section in Adjudication.

    Submittal Files

    If you wish to see the actual 5010 837 file submitted to the clearinghouse, click the arrow underneath the words Submittal Files. Select the date and time of the batch you wish to view.  

    A window will popup displaying the contents of the file.  It cannot be edited, only viewed and/or printed.

    ERN Files

    If you wish to see the actual ERN file received from the clearinghouse, click the arrow underneath the words ERN Files. Select the date and time of the batch you wish to view.  

    A window will popup displaying the contents of the file.  It cannot be edited.  It can be printed or exported to an external text file by clicking the EasyPrint button.
