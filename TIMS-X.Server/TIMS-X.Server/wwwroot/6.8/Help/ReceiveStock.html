    Receive Stock

    Any hearing aid model or accessory can be added to stock.   Earmolds cannot be stock items. Only models that are active and marked Available for Sale can be added as stock.  You can track all your models this way, or choose only certain ones that will be "stock" items.  To add new hearing aids or accessories to stock, complete the following steps:


    1. Each time you enter the Receive Stock link, the grid will be empty.
    2. The Date Received will default to the current date, but can be changed as needed.
    3. Click the Add New icon to start adding stock items.
    4. A new line will appear with required fields highlighted in yellow. First select a Manufacturer and then the associated model you want to add. 
    5. The Style will default from the model's default style from setup.  This can be changed if needed.
    6. Select the Site that the item will be available at by clicking in the field.  (this is a required field)
    7. If available, add the Serial Number, P.O. No., Invoice Number, by clicking in the field and typing the information.
    8. The Price and Cost will default to what has been entered in the Model Setup.  These can be changed as needed.
    9. Add any applicable Notes, if desired.
    10. Click on the Options button to select any available options for the model.
    11. To continue adding more items to stock, add a new line to the grid by either clicking the Add New icon  on the toolbar. You can also copy information from the previous line by clicking the Duplicate Line  button at the beginning of the row.  A new line will appear with all the same information as the previous one, with the exception of the serial number.
    12. You can remove a previously entered line by clicking the Delete Line button at the beginning of the row.

    1. When all the items for the Date Received have been entered, click the Save icon on the toolbar.  A message will ask if you wish to attach an invoice to the batch of items just entered.  

      1. Click Yes - the TIMS Document Capture System screen will popup. Document Type will default to "Stock Invoices". Follow the instructions in Attaching Documents to a Patient Record.  The document will be associated with each of the stock items in the batch, and can be viewed from the Stock Item screen. Once the document is saved, the items will be saved and received into stock as well.
      2. Click No - the items will be saved and received into stock with no document attached.
      3. Click Cancel - return to Receive Stock to edit the list of items.
