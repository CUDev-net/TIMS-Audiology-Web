    Recurring Appointments

    Patient appointments and non-patient schedule items can be set as recurring, if they occur at regularly scheduled intervals.  

    Note: an appointment must be set as Recurring at the time it is created.  Once saved, it cannot be later changed to a recurring type.


    1. Choose the starting date of the recurring appointment and make an appointment or schedule item as described in the previous sections, but before clicking OK:
    2. Click the Recurrence check box.
    3. The recurrence options will be brought forward as a separate sub-tab, next to the Notes field.

    1. Interval Settings - this is where you will choose how often and on what schedule the appointment will occur. Click the down arrow and choose the desired interval from the box. 

      1. Daily - choose every 1-30 days, every weekday, every Monday, Wednesday, Friday, or every Tuesday and Thursday.

     

      1. Weekly - choose which day(s) of the week and how often 

      1. Monthly - choose how often, and which day of the month 

      1. Yearly - choose which month, then which day of the month 

    1. End Settings - choose when you'd like the recurring appointment to stop - either after a specific number of occurrences or on a specific date.  Choose "No end date" if you want the appointment to occur indefinitely.

    1. Summary - a summary description of all the interval settings that have been selected for the appointment.

    1. Click OK to save or Cancel to abandon changes.  

    Edit an existing recurring Patient Appointment  

    Note: Once a recurring Patient Appointment has been created and saved, you cannot change the recurrence Interval and End settings. If you wish to change the recurrence settings, you should delete the remainder of the series and create a new one.


    1. Using the Months Panel, set the Appointment Grid to display the desired day. Double-click on the recurring appointment.  The Appointment Details will popup.
      1. You can ONLY change fields in the Details section.  You cannot change the Recurrence settings.
      2. Changing other details, such as Provider, Appointment Type, time, etc will only change the selected appointment in the series.  It will not be changed for other appointments in the series.
    2. If you need to make changes to remainder of the series, or just want to remove the remaining appointments, click the End Series Here button .  This will delete the selected appointment and remaining ones in the series.  You can then recreate a new series with the changes needed.
    3. Click Save or Cancel to abandon changes.
    4. Note - you can not drag a recurring appointment to a new time on the grid.  You must open the details as described above and change the time or duration there.

    Edit an existing recurring Non-Patient Appointment:

    Note:  Once a recurring Non-Patient Appointment has been created and saved, you cannot change the recurrence Interval settings. If you wish to change the interval settings, you should delete the remainder of the series and create a new one.  You can however change the End Settings.


    1. Using the Months Panel, set the Appointment Grid to display the desired day. Double-click on the recurring appointment.  The Schedule Item screen will popup.
      1. You can ONLY change fields in the End Settings section.  You cannot change the Interval settings.
    2. Click OK to save or Cancel to abandon changes.

    Delete an existing Recurring Appointment

    • Using the Months Panel, set the Appointment Grid to display the desired day. Select the recurring appointment.
      • Patient - click the Delete icon  on the toolbar. A confirmation message will popup. Click Yes to delete or No to cancel.  Only the selected appointment will be deleted.  None of the appointments in the series will be removed. 
      • Non-Patient - click the Delete icon  on the toolbar. A message will popup asking if you want to delete just the selected appointment or all the appointments in the series.  Choose one or the other, then click OK to delete or Cancel to abandon changes.
