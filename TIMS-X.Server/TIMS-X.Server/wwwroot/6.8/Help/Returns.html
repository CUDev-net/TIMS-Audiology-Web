    Returns

    Record information about a hearing device's return by clicking the Returned button .  Also, if you change the status to "Returned", the Hearing Aid Returned screen will popup.  Only Custom Ordered or Stock hearing aid histories can be designated as Returned.  If there is neither an order entered or a stock item assigned, the Returned button will be inactive and the "Returned" status unavailable.  Return procedures are the same for both Hearing Aids and Accessories. See Returns Workflow for an overview of the returns process.

    Return a Custom Order


    1. An order must be Received first before it can be returned.  The status cannot be "On Order".
    2. Click the Returned button , or change the status to "Returned".  

    1. The Hearing Aid Return screen will popup. Choose whether the device will be returned to Manufacturer or Stock.

    1. Fill in the Return Date and choose a Return Reason  - these are required fields.
    2. If Returned to Manufacturer is checked, the device will then be automatically transferred to Stock on Hand, with a note indicating which patient previously purchased it and when it was returned. Verify the Return to Site is correct and change if necessary. The P.O. No. and Serial Number will automatically be entered.  The device will be made Inactive and the stock status will be set to "Returned for Credit". If the order had a scanned document associated with it, that will be available to view in the Stock Item screen as well. Enter any necessary notes.  Click OK.
    3. If Returned to Stock is checked, verify the Return to Site and Return to Stock Status are correct and change if necessary. Click OK.  
      1. The Return For Resale screen will popup.  

      1. Click Return To Stock, or type an Inactive Reason and click Inactivate Item.  The device will then be automatically transferred to Stock on Hand, with a note indicating which patient previously purchased it and when it was returned.  The P.O. No. and Serial Number will automatically be entered.  If Return To Stock was clicked, the device will be under Active Stock Items and the stock status will be "Used".  If Inactivate was clicked, the device will be under Inactive Stock Items and the stock status will still be "New".  If the order had a scanned document associated with it, that will be available to view in the Stock Item screen as well. Enter any necessary notes.  
    1. A note about the return will automatically be put in the hearing aid history Notes field.  The status will be locked, and the order and return information (other than notes) can no longer be edited.  The Returned button will now be orange in color .  The status of the device is now "Returned", which is an inactive status.


    Return a Stock Item


    1. Click the Returned button , OR change the status to "Returned" OR click the Stock Item button .  The Hearing Aid Return screen will popup. Choose whether the device will be returned to Manufacturer or Stock.
    2. Fill in the Return Date and choose a Return Reason  - these are required fields. Enter any necessary notes. Click OK.
    3. If Returned to Manufacturer is checked, the Return For Resale screen will popup. 

      1. Click Return To Stock, or type an Inactive Reason and click Inactivate Item. The device will then be automatically transferred to Stock on Hand, with a note indicating which patient previously purchased it and when it was returned.  The P.O. No. and Serial Number will automatically be entered.  The device will be under Active Stock Items and the stock status will be "Used". If the device had a scanned document associated with it, that will be available to view in the Stock Item screen as well. Enter any necessary notes.  If Inactivate was clicked, the device will be under Inactive Stock Items and the stock status will be "Used". 
    1. If Returned to Stock is checked, verify the Return to Site and Return to Stock Status are correct and change if necessary. Click OK. The Return For Resale screen will popup.  Click Return To Stock, or type an Inactive Reason and click Inactivate Item.
      1. The device will then be automatically transferred to Stock on Hand, with a note indicating which patient previously purchased it and when it was returned.  If Return To Stock was clicked, the device will be under Active Stock Items and the stock status will be whatever was previously selected.  If Inactivate was clicked, the device will be under Inactive Stock Items.  If the device had a scanned document associated with it, that will be available to view in the Stock Item screen as well.
    2. A note about the return will automatically be put in the hearing aid history Notes field.  The status will be locked, and the stock item and return information (other than notes)  can no longer be edited. The Returned button will now be orange in color . The status of the device is now "Returned", which is an inactive status.
