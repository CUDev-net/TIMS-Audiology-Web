    SLP

    SLP (Speech-Language Pathology) users are able to configure appointments, charting and reporting in TIMS to pertain specifically to SLP functionality. Selecting an SLP appointment type cues TIMS to configure the history page with only SLP charting options and provides intuitive, user-friendly access to progress notes, assessment results and easy links to online assessment forms.


    SLP specific scripted notes allow quick customized templates to be added into the progress notes, diagnosis, recommendation and goals fields. Each of those fields can then be incorporated into customized letters and reports.  TIMS also offers streamlined integration of patient data into patient letter and report templates with the options of printing, faxing or emailing to multiple recipients. 


    Office Notes

    Additional notes that will not be printed on any Diagnostic reports.

    SLP

    If the History Type of the selected History record is an SLP specialty, this screen

    will automatically be shown by default. NOAH testing modules will be replaced with

    online SLP assessment modules.

       

    Copy (button)

    Click the Copy button to bring up the Copy SLP screen.  

     

    Choose the prior history session you wish to copy data from.  Choose the fields you wish to copy into the current session. If user preferences have been set to copy certain fields, these will be checked by default. Click OK to copy or Cancel to abandon changes.

    Chart Notes (button)

    Click the Chart Notes button  to see a summary of all history notes (including communication interactions) for the patient.  If Noah 4.2 or above is installed, this summary will also include the patient's audiograms, test results, telehealth sessions, medical conditions, medications, and current hearing aids.  The notes summary and/or audiograms can be printed from this screen. Click the down arrow next to Print to see the print options.

       

     

    Notes can be printed individually or all at once.  Click on the audiogram to print an individual diagnostic report for the doctor or patient.

    Developmental History

    Record the age in years and months that the patient achieved various developmental

    milestones.  Milestones can be added and/or edited in SLP Setup.

    Reports

    Choose from the available SLP reports that have been added through the report designer.  See Diagnostic and SLP Reports for more information.

       

    Behavioral Observations

    Select the appropriate evidence observed for the various listed behaviors.

    Communication Profile

    Record the level of various communication skills.

    Fluency/Voice

    Record the level of fluency/voice skills.

    Recommendations

    Select and enter notes for patient recommendations.  Recommendations can be added and/or edited in SLP Setup. Scripted Notes can also be used in this field.

    Diagnosis

    If specific Diagnosis codes have been selected in the Electronic Superbill for this

    History, they will automatically be copied here.  Otherwise, they can added as needed. Scripted Notes can also be used in this field.

    Goals

    Click the Edit button to add Goals for the patient.

     

    Once the goals have been added to the screen, choose the progress level - In Progress, Met, or Not Met.  Goals can be added and/or edited in SLP Setup. Scripted Notes can also be used in this field.

    Assessments

    Choose one of the included assessments to record the date of assessment and results.

     

    Notes

    Open fields for entering progress notes pertaining to the patient's visit.  An unlimited number of characters can be entered in each field.  You can type directly in the box for a plain text note, or if a more formatted note is desired, right-click on the text box and select Open Editor from the menu. 

       

    The Note Editor will popup.  Type your text and format as desired using the toolbar buttons.  You can also copy and paste text from another application. See Rich Text Editor for more information and Speech Recognition instructions.  


    Context Menu - right-clicking in each of the fields will display a context menu with

    the following choices:

    • Disable SpellChecker - if spell checker is enabled for this user, you can disable it here if not needed.
    • Undo, Redo, Cut, Copy, & Paste – These commands are tied to, and follow the standard behavior of, Windows System commands.  As a side note, when a form and its text box(s) are first loaded, the Undo function is not available until a change is made to the text.
    • Add to Dictionary, Ignore All, Replace  - if spell-checker is enabled and a word is identified as a misspelling, these choices will be displayed.
    • Open Editor - Use to input rich text formatted notes.   See Rich Text Editor for more information.
    • Enable Spell Checker - Click here to turn on the spell-checker if it has been disabled for this user in User Preferences.
    • Categories and Scripted Notes -  Use to insert standardized templates that you have entered as Scripted Notes in the History Setup. The Scripted Notes are organized/sorted by their associated Categories. Only the categories for the appropriate Field (Note Type) will appear (Diagnosis, Recommendation, Chart)  Once a category is selected, the available Scripted Notes for the Note Type and User will be displayed.  Select the desired note to paste it into the body of the field.
