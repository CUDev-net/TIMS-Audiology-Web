    Schedule Blocks

    Schedule Blocks are a way to designate certain times and days for particular appointment types and providers and provides the ability to search for open blocks of time based on a set of criteria.  The criteria may include provider, site, appointment type or a date range.

    Add a Schedule Block and Timeslots

    1. Click the Add New icon.
    2. Complete the following information:
      1. Name - this is the name that will appear in the Block Legend in the Appointment Link.
      2. Appointment Types - select all the appointment types that will be included in the block. Multiple appointment types can be assigned to a block.  However, an appointment type can only exist in 1 block. If you attempt to assign an appointment type to more than one block, a popup message will warn you that this is not allowed.

      1. Color - choose a color to represent the block. The color will appear on the calendar grid to designate the block times,  and also in the Block Legend in the Appointment Link.
      2. Notes (if desired)
    1. Click the Save icon to save the new schedule block.
    2. Once the block has been added, providers must be assigned hours to it, in order for the block to be displayed on the calendar.
    3. Highlight the desired block on the Schedule Blocks tree, and click the Add Timeslot button.  The Add Provider Time Slot window will popup.  Complete the following information:

      1. Provider - choose from active providers.
      2. Day - must be a day the provider is scheduled to work.  (see User Preferences)
      3. Start and End Time - the hours assigned must be within the provider working hours. A popup message will warn you if hours are entered that the provider is not scheduled for.

      1. Note - a provider can not be scheduled for more than one block at the same time.  If conflicting time slots are entered, a warning message will popup.

      1. Click OK to save, or Cancel to abandon changes.  The timeslot will appear in the Provider Timeslots panel to the right.


    1. Multiple time slots for the block and provider may be assigned, as long as they do not conflict with any other time slot.


    Only once all the above has been completed, will block scheduling be operational.  If a block is created, but no appointment types added to it, it is ignored.  If a block is created, but no providers have hours for it, it is ignored.

    Remove a Timeslot

    To delete a previously entered timeslot, select it on the Provider Timeslots panel and click the Remove Times button .  A popup message will confirm that you really wish to remove the timeslot.  Click Yes to remove, No to cancel.  

    Note - provider timeslots cannot be edited once they have been entered.  If a timeslot needs to be

    changed, simply delete it and add the new one.

    Schedule Block Report

    To see all the entered Schedule Blocks, on the menu click Reports - Appointments - Schedule Blocks.  The report will give options to select certain Providers or Blocks only, and display everything that has been previously setup, including the colors.
