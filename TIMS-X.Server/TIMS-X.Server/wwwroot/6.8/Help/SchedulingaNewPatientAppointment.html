    Scheduling a New Patient Appointment

    Search for an Opening

    1. Click on the Search for Appointment icon in the top part of the screen.  The search criteria screen will popup. 

      1. Providers - Check the name of the provider for whom you wish to search for openings. Check All Providers to search all of them.  
      2. Sites - Check the name of the site for which you wish to search for openings.  Check All Sites to search all of them.  
      3. Resources - Check the name of the resource for which you wish to search for openings. Individual Resources are only shown when a specific Site is checked, and are not shown when All Sites are checked. Check All Resources to search all of them.  
      4. Specialties - Check either All Specialties, or just Audiology or SLP.  The Appointment Type list will be filtered accordingly.
      5. Change the Start and End search date, if needed.  If not changed, the search period will default to 1 month, starting from today's date.
      6. Select an Appointment Type, if desired.  Selecting an appointment type sets the duration time to the default for that type, as set in the Setup Link. The duration time can be changed manually if needed.   Note: If the appointment type selected is contained in a  Schedule Block, only previously setup time slots for that type/block will be searched. 

             

      1. If the Duration has not been defaulted from the appointment type, change to the desired amount of time, from 1 minute to 11 hours 59 minutes.
    1. Click Find Openings to see a list of times that match the selected parameters, or Close to cancel the search. 
    2. A list of available openings will be displayed.

    1. Search for the patient name on the tree.  If the patient is not found in the database, add as new.  See Patient List Panel for details. 
    2. From the Patient Tree, select the patient name, hold down the left mouse button, drag the name and drop it on the desired time slot.  

    1. The Appointment Details pop-up screen will appear. 

      1. Patient name and Phone will be copied from the patient's information. If the name of the patient is red, that means there are Communication Restrictions that will appear in the Notes section.
      2. Opportunity status will be calculated and displayed automatically. The status is based on a combination of factors from hearing aid history, NOAH test results, and communication restrictions and shows the potential for future hearing aid sales.  See Opportunity Tracking for more details.
      3. Provider will default in the following order:
        1. The provider name at the top of the column, if more than one provider is chosen and the Providers View is selected. 
        2. The provider selected in the "Seen By" field in the patient's details.
        3. The provider field will be blank, and you must choose one before saving the appointment, if none of the above conditions are true.
      4. Appt Type will default to the type searched for, if one was entered.  If no type was entered, it will be blank - click on the down arrow to select one. Appointment types can be added and changed in the Setup Link.  This must be done before the appointment is added, however.
      5. Site will default in the following order:
        1. The site name at the top of the column, if more than one site is chosen and the Site View is selected.
        2. The patient's default Site as entered in the patient's details.
      6. Resource will default in the following order:
        1. The resource name at the top of the column, if more than one resource is chosen and the Resources view is selected.
        2. The field will be blank and you can choose one if desired. 
      7. Status will default to "New", but can be changed as needed.  Appointment statuses can be added and changed in the Setup Link.
      8. Marketing - If this is the first appointment for the patient, and a marketing source was set in the Patient details, that source will default here.   It can be changed if needed.  Otherwise, a source can be added if desired (or required).  Click the ellipsis button to see a popup of all the available choices. 


    Click the arrow symbol  next to the desired category, then double-click on the source to select. Marketing categories and sources can be added in the Setup link.

      1. Authorization  - If appointment authorizations have been enabled, and the patient has active authorizations available, this field will display the name and number remaining. 

    If the authorization is selected, the number used count will increase by one. For a recurring appointment series, one authorization will be applied to each member until all have been used, or the expiration date has been reached.  Authorizations will become inactive when number of appointments has been reached or expiration date has expired. Inactive authorizations will no longer be displayed.

      1. Date will default to the date of the slot on the appointment grid where the patient name was dropped.  If this date  is not correct, you can change it here if needed.  The date can be changed by clicking on the calendar icon and selecting a new date.

      1. Time will default to the beginning time of the slot on the results list where the patient name was dropped.  If this time is not correct, you can change it here if needed. The start time can be changed by clicking on the time and typing a new value.  The end time will be calculated automatically by adding the duration to the start time.   
      2. Duration will default to the length set for the Appointment Type in the Setup link. It can be changed if needed by clicking on the arrows or typing the desired minutes and hours.  You can enter any value from 1 minute to 11 hours and 59 minutes.
      3. Next Contact will default to blank.  If you wish to set a date to contact this patient next, enter it here. The date can be added by clicking on the calendar icon and selecting a date.
      4. Take Photo - if a picture of the patient has previously been obtained, it will appear here.  If you have a web camera attached to your computer you can add or change the picture by clicking Take Photo.  Follow the instructions in Taking a Patient Picture.
      5. Scan Document - if you have a scanner attached to your computer, you can scan in any document that you want to associate with this patient.  If no scanner is attached, you can also attach any previously scanned image or PDF file.  Click the Scan Document button to open the TIMS Document Capture System screen.  Follow the instructions in Attaching Documents to a Patient Record.
      6. Digital Intake - Click here to generate a message to the patient (Email and/or Text).  The message will contain a link for the patient to click on.  This will take them to the online digital intake sheet where they can submit their demographic information.

      1. Add to Cancellation List - if the patient would prefer an earlier appointment than the one scheduled, check this box.  Then if an opening comes available, a popup list will display the names and details of all the patients who would like to fill that time slot.  See Appointment Cancellation List for details.
      2. Recurrence - if the appointment is a recurring one, check the Recurrence box, and follow the steps in the Recurring Appointments section.
      3. Notes - if the patient has any Communication Restrictions entered in the patient details, those restrictions will appear here.  Otherwise this field will be blank.  You can add any notes you wish, even if the restrictions are already in there - up to 255 characters.
      4. Tracking - Created, Updated, By Site, and By User - The Created field will show today's date.  The other fields will be blank when a new appointment is made.  They will be filled in automatically after the appointment is saved.
    1. When all the appointment information is entered, click the OK button . 
      1. If you click Cancel , the appointment will not be saved and you will lose the information that you entered.  The search results list will close, and the appointment will be displayed on the grid.

    Additional Notes

    • If you already know the day and time on which you want to schedule the appointment, simply switch to that day by clicking on the appropriate date on the Months Panel.  Then proceed from Step 4 above, dragging the patient name onto the desired time.
    • Multiple patients can be scheduled for the same time slot.  To schedule multiple patients for one time period, simply drag and drop them in the same time slot on the grid.  Color-coding by provider will show you if you have accidentally booked more than one appointment for the same provider at the same time.
    • If you make an appointment for any date and time before the current date and time - a popup message will inform you of this, and ask if you're sure that's what you want to do.

    • When an appointment is made for a patient, an associated History session will also be created at the same time, if this option has been checked in Practice Setup.  The linked History session can be viewed on or after the date of the appointment.
    • If you make an appointment for a provider for a site or time that the provider is not scheduled to be available, a popup message will inform you and ask if you want to continue.

    • If you make an appointment for an appointment type in a block timeslot that is designated for a different type, a popup message will inform you and ask if you want to continue.
