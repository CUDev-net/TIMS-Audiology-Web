    Scheduling a Non-Patient Appointment

    You can schedule a non-patient appointment, otherwise known as a "Schedule Item".  A Schedule Item is still assigned to one or more providers, but not to a patient.  This type of appointment can be used for meetings, lunches, out of office, etc.

    Search for an Opening

    1. Click on the Search for Appointment icon in the top part of the screen.  The search criteria screen will popup. 

      1. Providers - Check the name of the provider for whom you wish to search for openings. Check All Providers to search all of them.  
      2. Sites - Check the name of the site for which you wish to search for openings.  Check All Sites to search all of them.  
      3. Resources - Check the name of the resource for which you wish to search for openings. Individual Resources are only shown when a specific Site is checked, and are not shown when All Sites are checked. Check All Resources to search all of them.  
      4. Specialties - Check either All Specialties, or just Audiology or SLP.  The Appointment Type list will be filtered accordingly.
      5. Change the Start and End search date, if needed.  If not changed, the search period will default to 1 month, starting from today's date.
    1. Click Find Openings to see a list of times that match the selected parameters, or Close to cancel the search. 
    2. A list of available openings will be displayed.

    1. Double-click on the desired time slot. The Create New Schedule Item screen will open. 

      1. The Provider(s) field will automatically be filled in with the name of the provider you clicked on.  You can select different, additional, or all providers by clicking the down arrow on the field and checking the box next to the desired name(s). This is a required field.
      2. The Sites field will automatically be filled in with the name of the provider you clicked on.  You can select different, additional, or all providers by clicking the down arrow on the field and checking the box next to the desired site name(s). This is a required field.
      3. Type a brief description of the appointment in the Subject field. This is a required field.
      4. Date will default to the date of the slot on the results list, that was double-clicked.  If this date  is not correct, you can change it here if needed.  The date can be changed by clicking on the calendar icon and selecting a new date.  

      1. Time will default to the beginning time of the slot on the results list, that was double-clicked.   If this time is not correct, you can change it here if needed. The start time can be changed by clicking on the time and typing a new value.  The end time will be calculated automatically by adding the duration to the start time.
      2. Set the Duration to any time from 1 minute to 11 hours 59 minutes.  
      3. Type a brief description of the Location for the appointment, if desired.  This can be simply the office name or something more descriptive, such as "conference room", "John's office", etc.
      4. Choose a specific Color (if desired) by clicking on the color button . The appointment will appear on the grid using this color with a striped background.  

      1. Type any Notes, if desired.
    1. Click OK to save or Cancel to abandon changes.
    2. If the schedule item is a recurring one, check the Recurrence box, and follow the steps in the Recurring Appointments section.

    Additional Notes: 

    • If you already know the day and time on which you want to schedule the appointment, simply switch to that day by clicking on the appropriate date on the Months Panel.  Then proceed from Step 4 above, double-clicking on the desired timeslot.
    • If you make an appointment for any date and time before the current date and time - a popup message will inform you of this, and ask if you're sure that's what you want to do.

    • If you make an appointment for a provider for a site or time that the provider is not scheduled to be available, a popup will inform you and ask if you want to continue.
