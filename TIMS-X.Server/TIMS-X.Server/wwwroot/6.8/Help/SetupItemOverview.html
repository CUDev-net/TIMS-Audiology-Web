    Setup Item Overview

    The Setup link is where the various setup items that are used can be added, deleted, and edited. When you select a Category in the tree, you will see sub-links for the various items in that category.  In the sub-link, highlight an item in the tree and the fields will show to the right of the tree. Here the fields can be changed or new entries can be added or deleted.


    Adding an Item 

    1. Click the Add New icon on the toolbar. A blank item template will appear.
    2. Enter the information into the item fields and choose the Save icon on the toolbar.


    Deleting an Item from the Setup Link


    Items can only be deleted if they have not been used in any record.  For example, if an Appointment Type has been used in an appointments, then you will not be able to delete it.  If you wish to no longer use that type, you must mark it as Inactive instead. A Protected item cannot be deleted or inactivated.


    1. Select the item you wish to delete on the tree.
    2. Click the Delete icon on the toolbar.  If the item is in use, a message will popup informing you of that, and the item cannot be deleted.
    3. A message will popup confirming that you wish to delete the item.  Click Yes to delete, or No to discard your changes.
