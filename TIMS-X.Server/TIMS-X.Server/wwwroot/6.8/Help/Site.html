    Site

    If you have multiple locations for your practice, you can enter them here.    

    Note:  You must have at least 1 Site entered, because a Site is required for all Patients and Appointments.


    Name

    The name of the site/location.

    Description

    Description of the location if needed.

    Address1 - Zip Code

    The address for this location, for use in the printing of forms. This is the address that will appear on reports if "Use Site Address for Reports" is checked in Practice setup. It is also the address used for Service Facility Location (box 32) on claim forms. Click on the Verify Address button to lookup and confirm the official address.

    Phone

    The main billing and correspondence phone number of the location.

    Fax

    The fax number  for this location, for use in faxing of documents

    NPI

    The National Practitioner Identifier number - a unique identifier for health care

    providers.   To use the NPI lookup tool, click on the magnifying glass icon . The NPI Lookup Form will popup. The First, Last, City, State and Zip Code of the physician will be pre-populated if that information has already been entered.  Change the search fields as needed and click Search. Double-click on the desired match to enter the NPI found.     

     

    Default Tax Group

    If you are using the POS module, choose the default tax rate for this location.

    Color

    Select the color associated with the site.  If desired, appointments can be color-coded by this on the appointment grid.  See User Preferences.

    Site Logo

    Click the Site Logo button to choose a logo image file for use with letters and

    reports. Browse to and select the image file, then click Save.

     

    A message will ask if you wish to apply the logo to all sites.  Click Yes for all sites, No for the selected one only.

     



    Office Hours

    The starting and ending hours that the site is available for appointments for each

    day of the week.

    Credit Card Processing

    Click the ellipsis button  to open the E-Payment Gateway Setup.  See E-Payment Processing for details.

    E-Check Processing

    Click the ellipsis button  to open the E-Payment Gateway Setup.  See E-Payment Processing for details.

    CareCredit

    Information associated with the location's CareCredit™ financing registration. 

    Transnational API 

    Auth Key

    Information associated with the locations Transnational (credit card processing)

    registration.

    AllWell ID

    The ID associated with the locations AllWell™ Financing registration.

    Caption Call Promo Code

    Promo code to be used with submitted CaptionCall™ applications.

    Outreach Educator (CapTel)

    Default outreach educator name to be used with submitted Hamilton CapTel™ applications.

    Misc Text 1, 2, and 3

    Free form text fields to be used as needed by the Practice.  Most often used to record an ID or value needed for a particular form.  Not used for any claim purpose.

    Use with Patient Scheduling

    This box will only be visible if Online Patient Scheduling has been enabled for the practice.  Check this box to make this Site available online for patient's to choose when requesting an appointment.

    Inactive

    Check this box if the site will no longer be used.
