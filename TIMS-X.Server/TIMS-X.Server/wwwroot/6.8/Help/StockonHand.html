    Stock on Hand

    Once items have been added to stock in the Receive Stock tab, they can be viewed, edited, inactivated, or deleted in the Stock on Hand sub-tab, as long as they have not yet been assigned to a patient in HA History. Once an item has been assigned to a patient, it will no longer appear in this list. (see Add a New Hearing Aid History)


    The available stock items are listed in the Stock Items tree, sorted by Manufacturer. 

    The item name is followed by the serial number, if one has been entered.  If the item is an accessory, and (A) will appear at the end. Inactive stock items will appear at the bottom.  Beginning in version 6.07, you can now search for a stock item by serial number. Type all or part of the serial number in the search box, then click the magnifying glass icon. 

    After selecting an item from the tree, the details can be viewed on the screen to the right.  

    If needed, you can edit the PO Number, Invoice Number, Serial Number, Cost, Date Received, Site, Status, or Notes.  You can also set a stock item to be inactive - which requires a reason to be entered.  Inactive stock items can not be assigned to a patient.


    Manufacturer

    The manufacturer of the item, as selected in the Receive Stock tab.  This field cannot be changed.

    Model

    The model name of the item, as selected in the Receive Stock tab.  This field cannot be changed.

    Style

    The style of the model, as selected in the Receive Stock tab. When assigned to a patient, this price will be entered into the HA History record by default.

    Price

    The price that the item will be sold for. When assigned to a patient, this price

    will be entered into the HA History record by default.

    Cost

    The cost of the item.  If this was entered in Receive Stock, it will be displayed

    here.  Otherwise it can be entered or changed if needed.

    Site

    The site where the item is located, as selected in the Receive Stock tab.  It can be changed here if needed.

    P.O. No.

    If Purchase Order number used for the purchase of the item.  If this was entered in Receive Stock, it will be displayed here.  Otherwise it can be entered or changed if needed.

    Invoice Number

    The invoice number for the purchase of the item.  If this was entered in Receive Stock, it will be displayed here.  Otherwise it can be entered or changed if needed.  Some invoice numbers will auto-generated if the stock item was automatically entered because of a customer return.  For example, if a patient returns a custom ordered hearing aid, it will automatically be placed into stock, before being returned to the manufacturer. The Invoice Number will show "HA Return", plus the original Order Number for the item.

    Serial Number

    The serial number of the item.  If this was entered in Receive Stock, it will be

    displayed here.  Otherwise it can be entered or changed if needed.

    Date Received

    The date the item was received into stock, as entered in Receive Stock.  It can be changed here if needed.

    Notes

    A free form field for entering any notes associated with the item.  If this was entered in Receive Stock, it will be displayed here.  Otherwise it can be entered or changed if needed. Note: Some notes will auto-generated if the stock item was automatically entered because of a customer return.  For example, if a patient returns a custom ordered hearing aid, it will automatically be placed into stock, before being returned to the manufacturer.  A note such as "Custom order returned for resale from Patient Robert Erickson" will appear.

    Status

    When an item is Received into Stock, the default status will be "New". Standard statuses are "New", "Used", "Returned for Credit", "Returned for Repackaging", "Rebuilt", and "Damaged".  Custom statuses can be setup as needed. Certain statuses will automatically be set when an item is placed into stock because of a customer return. For a custom order or stock item return, the status will be set as "Used".  For a custom order that was marked as "Returned to Manufacturer", the status will be set to "Returned for Credit".  Only stock items with a status of "New" or "Used" can be issued to a patient.

    Inactive

    Check this box to inactivate a stock item, so it will not be available to sell.  A

    reason is required when Inactive is checked.  When an item is returned in HAHistory, it can also be made inactive at that time.

    Reason

    A free form field to enter the reason the item was made inactive.  This is required when the Inactive checkbox is checked.

    Options

    Optional add-ons for this item.  When assigned to a patient, these options will be entered into the HA History record by default.

    Scan Document

    If you wish to scan in a document to be associated with the stock item, click the Scan Document button.  The TIMS Document Capture System screen will popup. Document Type will default to "Stock Invoices". Follow the instructions in Attaching Documents to a Patient Record.  The difference is that the document will be associated with the stock item, and can only be viewed from the Stock Item screen or the Imaging Center.  Once the document has been saved, the button will then read "View Document".  Click on the button to see the previously saved document.  If a stock item is a returned custom order, any documents previously associated with the order will be available here.  If a document was attached at the time the stock item was received, the button will also read "View Document".

    Updated

    The date when the stock item information was last updated. This is automatically updated when the record has been changed and saved.

    By User

    The last User to update this stock item record.

    Delete Stock Item

    To remove an item from stock completely, select it on the tree and then click the delete icon .  A message will ask you to confirm the deletion.

     Click Yes to delete, or No to cancel.
