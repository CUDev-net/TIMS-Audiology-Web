    TIMS POS Module

    The Point-Of-Sale (POS) Module provides a quick way for users to produce and print a point-of-sale document from within TIMS. These documents include invoices, credit memos, payment receipts, refunds, and insurance transfers. All of these documents can be tracked with TIMS Accounts Receivable or exported to QuickBooks®. 


    Automated invoices in POS can be created for entries made in Hearing Aid History, Repairs, and Battery Club. For example, if you create and save a new Hearing Aid History for a patient and check the "Invoice" box, you can immediately create an invoice for that hearing aid by running the POS module. The hearing aid details (manufacturer, model, serial number, default price, etc) will be transferred to the invoice. You can then change the price and/or add any other items, discounts, comments, or a payment if you wish.


    The POS Module can accommodate multi-site users. Remote locations can generate POS documents and receive payments. The site name selected will be saved with the document. After the database is replicated, any office is able to see documents any other given office has created. A POS document can be generated for any patient at any location, regardless of what their default site may be. For instance, a patient could purchase a hearing aid at Site A and later drop off a payment at Site B. Both transactions would appear in the patient's Activity list (in TIMS) and would be exported to the main QuickBooks® database (if utilizing that feature). The document's site will be exported to QuickBooks® as a "Class", allowing for sales reports that can be sorted or filtered by location.
