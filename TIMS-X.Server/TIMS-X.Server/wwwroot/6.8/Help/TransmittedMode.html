    Transmitted Mode

    In this mode, claims that have been transmitted to the clearinghouse, but have not yet received a response, are displayed on the patient tree.

    Patient Tree

    Expanded:

    Collapsed:

    If the tree is collapsed, expand by clicking on the  arrow symbol .  

    1. The First Level of the tree will display the Patient Name.  
    2. The Second Level displays the Claim Number,Transmit Date, Batch Number, Appointment Site, and a "P" (professional claim) or "I" (institutional claim). If the claim information is in purple font, this means that claim attachment information was entered with the claim before it was transmitted. See Attachment below for sending the attachment through TIMS.

    Select a claim line to see the details in the Claim Information section to the right.  This information can not be edited for transmitted claims.

    Action Buttons

    The following buttons appear on top of the tree when in Transmitted mode:

    • Expand - expands all nodes on the tree, so that all levels are displayed for all patients.
    • Collapse - collapses the tree to a condensed state, so that only the top level (patient name) is displayed
    • Select All - places a check mark next to all transactions in the tree
    • UnSelect - removes the check mark from all transactions in the tree
    • Revert - moves the selected claim(s) back to selected claims mode, so that it can be transmitted again.

    Processing Buttons


    • ClearingHouse - view previously received claim responses and download a response file without first transmitting a batch. See Reading Response Files and Download Response Files for more details.

    • View SuperBill Notes - if the transaction was initially created on an  Electronic Superbill, any notes entered in that screen can be viewed here.
    • View Payer Notes - if the transaction has been previously submitted, any previous clearinghouse and payer claim status notes can be viewed here. Additional manual notes can also be added in this screen. See Payer and User Notes for more information.

    • Attachment - if claim attachment information was entered with the claim before it was transmitted, this button will be active, and the claim will be in purple font on the tree. 

    Click here to attach and send the document to the payer.  The payer must previously be setup to receive attachments via fax, email or electronically (X12-275) and the appropriate fax and/or email address entered.

    Select the desired document to send to the payer. 

    Documents saved under Patient Documents will be available to select.  The default document type is Claim Attachment, but any document type can be selected.

      1. If the document has not been specifically requested by the payer yet, check the Un-Solicited box. If it has been requested, check Solicited and enter the identification number provided by the payer. 

      1. Optional - if you wish to add any additional comments to the payer, enter them on the Note on Cover page.

      1. Click the Preview File button to preview both the cover page and document that will be sent to the payer.

      1. If the cover page and document look correct, click the Fax, Email, or Electronic button to send the document via that method.
