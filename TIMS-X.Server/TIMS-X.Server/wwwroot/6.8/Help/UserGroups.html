    User Groups

    User Groups allow specific control over access to sensitive patient information.


    • Click the Add New icon on the toolbar to create a new User Group.  Type a name, then click Save.
    • Click the Edit button underneath the Users box.  Choose the users that will be members of the new group. Click Save.
    • Click the Edit button underneath the Application  Settings box.  Choose the permissions that members of the group will have.  Click Save .
    • To view and print all the current permissions for each user, run the User Permission List report, found under TIMS Reports-Management.
