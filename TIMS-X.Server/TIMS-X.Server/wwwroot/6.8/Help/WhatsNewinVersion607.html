  1. What's New

    What's New in Version 6.07

    Patients 

    • Privacy Mode - a new display mode has been added that will only show patient's initials instead of their full name. Patient photos will also be hidden. Click the new Privacy Mode icon (available in any link in the upper right corner) to enable or disable this mode. (see Privacy Mode for more information)
    • Two-way Text Messaging - a two-way text conversation can now be initiated directly from the Patient Summary Panel by clicking the new Conversations button. Practice must be enrolled in Automated Appointment Notification service, contact TIMS Support at 800-763-0308 for more information. (see Patient Text Conversation for more information)
    • Custom Fields - additional custom text and date fields (as many as you need) can now be added to the patient details. See Custom Fields setup for more information.
    • Required Fields - a practice can now designate which fields should be required in the patient details. These fields will be highlighted in yellow, and a patient cannot be saved until the required fields are filled out. See Practice setup for details.
    • Patient Summary
      • Details for the patient's Referring and Primary Care physicians can now be viewed in the Patient Summary (practice, address, phone, and fax numbers). 
      • The patient's latest audiogram can now be viewed in a popup tooltip when hovering the mouse of the Last History date.
      • If the practice is enrolled in Automated Appointment Notification service, you can now send an email via TIMS Email Service. This email will be archived and displayed in the Patient Activity panel under Communication Interactions.
      • See Patient Summary Panel for details.
    • Digital Intake Sheet - replicated and cloud practices can now send a link to a patient via text or email that will allow them to fill out intake information online. The submitted information then can be reviewed and merged to the patient's record in TIMS.

    New Setup Link 

    • Setup for all categories is now conveniently located in one link. See Setup Item Overview for more information.

    Appointment Notifications

    • Multiple Language support - templates can be created in a large selection of languages, and will automatically be sent to patients with the matching language. Note: the template text must be entered in the desired language, TIMS does not do any actual translating.
    • Multiple Voice support - there is now a large selection of voices that can be selected for voice (phone call) templates.
    • Preview - listen to voice mail, send test texts and emails when creating or editing templates.
    • Notification Center improvements - new filters for Appointment Type, Site, and date Created. The appointment list has an improved layout, along with better error reporting.
    • Appointment Summary displays notification results.
    • See Appointment Notification Overview for more information.

    Appointments

    • Appointment status can now be assigned an icon that will be easily visible on the grid. (No need to hover over the appointment to see the tooltip).
    • Appointment status can now be quickly set by right-clicking on the appointment in the grid.
    • Appointment status can now be set to receive alerts. 
    • Authorizations can now be applied across recurring appointments.
    • A new calendar interval of 20 minutes has been added.
    • Customize loading of appointments to improve speed - the number of months for future and past appointments can now be customized.

    Opportunity Status improvements

    • Patient and future appointment opportunity status is now updated automatically whenever an event takes place that will affect it. There is no longer a need to generate or refresh the OT Status for all patients in the Communications or Data Analysis links.

    Hearing Aid History

    • Repair Warranty Date - now has the option to be copied to the hearing aid history warranty date.
    • Repairs can now be copied to the other side of a binaural hearing aid history.
    • Payable Repairs can now be linked to a specific Repair POS item. When they are auto-invoiced, the proper service item will be billed. 
    • Returns - improved screen distinguishes between returning to manufacturer or to stock.

    Patient Documents

    • Mobile Uploading - practices whose databases are hosted with TIMS can now upload patient photos and insurance cards directly from their mobile device by scanning a QR code. (see Mobile Uploading of Photo or Insurance Card for more information)
    • Multiple document types can now be selected when scanning or uploading multi-page documents.

    Fax improvements

    • Multiple Fax numbers for physicians - in addition to the primary fax number, multiple additional fax numbers can be entered. These can be designated in order of preference to be used when faxing documents.
    • A one-time fax number can now be used, in addition to choosing from multiple fax numbers per physician.
    • The fax cover sheet can now be previewed before sending.
    • See Faxing Reports for more information

    Stock on Hand

    • Stock items can now be searched for by serial number.

    Chart Notes Printing improvements

    • Better options and layout of screen.

    Communications

    • Additional filters for previous medical history, preferred language, and appointment site have been added. Additional bookmarks have also been added for creating letters.

    TIMS Statements

    • Name and address for invoicing fields are now reflected in printed statements.

    Claims

    • Transactions link now has a "Group Add" function to quickly add multiple transaction lines at once.
    • Claim attachments - supporting documents for claims can now be sent directly through TIMS, via fax, email or electronically (X12-275).

    Auditing

    • TIMS can now be configured to track user's usage information on multiple tables in the database.  Depending on the table, information tracked includes User name, type of operation, date and time, Patient ID and name if applicable, Site, and more. The audit information can be viewed for a selected date range in the new Audits tab in the Centers Link. Contact TIMS Support at 800-763-0308 for more information.
